== Self-Grounding Agents with MAF and Neo4j

AI agents are powerful, but they share a set of common problems:

* **Hallucinations.** Without access to real data, agents fabricate answers that sound plausible but are wrong.
* **Lack of context.** Agents have no awareness of your domain knowledge, business data, or the relationships between entities unless you explicitly provide it.
* **Lack of memory.** Every conversation starts from scratch. Agents forget user preferences, past interactions, and previously learned facts.
* **Unreliable retrieval.** Most frameworks treat knowledge retrieval as a tool the agent decides to call. The agent may skip the tool, not realize it is relevant, or simply answer from its training data alone.

A **self-grounding agent** solves these problems by automatically receiving relevant knowledge from a graph database on every interaction, without relying on the agent to decide when to retrieve it. The Microsoft Agent Framework (MAF) makes this possible through **context providers**, a first-class mechanism that runs before and after every LLM call to inject knowledge and extract state.

This course teaches you how to build self-grounding agents using MAF and Neo4j. You will start with the fundamentals of MAF, learn how context providers work, then connect them to Neo4j for automatic knowledge graph retrieval with vector, fulltext, and hybrid search. Finally, you will give your agents persistent memory so they remember conversations, extract entities, and learn from past reasoning.

This course uses models from link:https://openai.com/[OpenAI^], although you can use the model and supplier of your choice.

=== Prerequisites

Before taking this course, you should have:

* A basic understanding of Graph Databases and Neo4j
* Knowledge of Python and basic familiarity with async programming

To complete the practical tasks within this course, you will need an link:https://platform.openai.com[OpenAI API key^]. If you are attending the **Road to NODES AI** workshop, an OpenAI API key will be provided for you.

We recommend taking the following courses:

* link:https://graphacademy.neo4j.com/courses/neo4j-fundamentals/[Neo4j Fundamentals^]
* link:https://graphacademy.neo4j.com/courses/genai-fundamentals/[Neo4j & GenAI Fundamentals^]

=== Duration

2 hours

=== What you will learn

* What self-grounding agents are and how context providers enable them
* How the Microsoft Agent Framework differs from other agent frameworks
* The context provider lifecycle (`before_run` / `after_run`) and how it differs from tools
* How to build custom context providers that inject dynamic instructions
* How to use `Neo4jContextProvider` with vector, fulltext, and hybrid search modes
* How to add graph-enriched retrieval using custom Cypher traversal queries
* How to give agents persistent memory with entity extraction, reasoning traces, and graph algorithms
