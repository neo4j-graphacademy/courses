= Introduction to Microsoft Agent Framework

This module covers what the Microsoft Agent Framework is, how it differs from other agent frameworks, and how to build your first agent with a tool that queries Neo4j.

In this module, you will:

* Learn about the Microsoft Agent Framework and its core concepts
* Set up your development environment with OpenAI and Neo4j
* Build your first agent with a Neo4j tool
