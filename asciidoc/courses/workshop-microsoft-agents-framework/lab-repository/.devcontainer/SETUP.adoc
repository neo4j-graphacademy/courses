= Setup

This repository accompanies the link:https://graphacademy.neo4j.com/courses/genai-maf-context-providers/[Neo4j Context Providers for Microsoft Agent Framework workshop^] on link:https://graphacademy.neo4j.com[GraphAcademy^].

All required software and packages have been installed automatically.

== Check Your .env

If you entered your credentials as Codespaces secrets when launching, your `.env` file has been generated automatically.

If not, create one manually:

[source,bash]
----
cp .env.example .env
----

Then update it with your Neo4j Sandbox and OpenAI credentials.

== Verify Your Setup

Run the test script to confirm your connections are working:

[source,bash]
----
python genai-maf-context-providers/test_environment.py
----

== Load Embeddings and Create Indexes

[IMPORTANT]
====
You must run the setup script before starting Labs 3+. It loads movie plot embeddings and creates the vector and fulltext indexes that the labs depend on. Without this step, you will get "index not found" errors.
====

[source,bash]
----
python genai-maf-context-providers/setup.py
----

You should see `Setup complete!` when it finishes.

== Start the Labs

The workshop labs are Jupyter notebooks in the `labs/` directory. Open `labs/lab-1-first-agent.ipynb` to get started.

=== Running the Notebooks

When you open a notebook, you need to select a Python kernel:

. In the top right corner, click **Select Kernel**
. Choose **Python Environments**
. Select **Python 3.12.11**

You are now ready to run cells in the notebook.
