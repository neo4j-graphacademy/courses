= Setup Your Development Environment
:order: 1
:type: lesson

In this workshop, you will create a Neo4j Sandbox, then launch a GitHub Codespace with your credentials pre-configured.

== Create a Neo4j Sandbox

You will need a Neo4j instance with the **recommendations** dataset. That dataset is a movie database with actors, directors, genres, and user ratings. The sandbox includes vector indexes on movie plot embeddings that you will use for semantic search in later labs.

. Go to link:https://sandbox.neo4j.com[sandbox.neo4j.com^]
. Sign up for a free account or sign in if you already have one
. On the next screen, you will see **Select a project**. Scroll down and choose the **Recommendations** dataset:
+
image::images/recommendations.png[Select the Recommendations dataset]
+
. Click **Create** to launch your sandbox
. Once created, download your credentials and save them somewhere safe. You will need the **Bolt URL**, **username**, and **password** in the next step

IMPORTANT: Save your sandbox credentials now. You will not be able to retrieve the password later.

== Launch GitHub Codespaces

IMPORTANT: The Codespace takes **3 to 5 minutes** to fully launch and configure. Wait until the `.env` file is created before trying to run the labs. You can click on the `.env` file in the Codespace file explorer to confirm it is populated with your credentials. Once you see your values there, you are ready to go.

Now that you have your Neo4j credentials, open the workshop repository in GitHub Codespaces:

link:https://codespaces.new/neo4j-partners/genai-maf-context-providers[image:https://github.com/codespaces/badge.svg[Open in GitHub Codespaces]^]

When you launch the Codespace, you will be prompted to enter your credentials as **secrets**:

OPENAI_API_KEY:: Your OpenAI API key from link:https://platform.openai.com[platform.openai.com^], or the key provided by your workshop instructor
NEO4J_URI:: The Bolt URI from your Neo4j Sandbox (e.g. `bolt://xx.xx.xx.xx:7687`)
+
NOTE: Both `bolt://` and `neo4j+s://` URI schemes work with the Neo4j driver. The Sandbox provides a `bolt://` URI. If you are using Neo4j AuraDB, you will have a `neo4j+s://` URI instead.
NEO4J_USERNAME:: Your Neo4j username (usually `neo4j`)
NEO4J_PASSWORD:: The password from your Neo4j Sandbox

Once the Codespace starts, it will automatically:

* Install Python 3.12 and all required packages (including `neo4j-agent-memory`, `openai`, `neo4j-graphrag`, and Jupyter)
* Generate your `.env` file from the secrets you provided

NOTE: If you need to manage your Codespace or secrets (for example, to relaunch with new credentials or shut down an existing Codespace), visit link:https://github.com/codespaces[github.com/codespaces^].

== Running the Notebooks

Each lab in this workshop is a Jupyter notebook. To run a notebook:

. Open the notebook file in the Codespace
. In the top right corner, click **Select Kernel**
. Choose **Python Environments**
. Select **Python 3.12.11**

You are now ready to run cells in the notebook.

== Verify Your Setup

Run the test environment script to verify everything is configured correctly:

[source,bash]
----
python genai-maf-context-providers/test_environment.py
----

You should see confirmation that your OpenAI and Neo4j connections are working.

== Load Embeddings and Create Indexes

[IMPORTANT]
.Run the setup script before you start the labs
====
You **must** run the setup script before starting the labs. This script loads pre-computed movie plot embeddings into your Neo4j Sandbox and creates the vector and fulltext indexes required by Labs 3 onwards. Without this step, vector and fulltext searches will fail with an "index not found" error.
====

Run the setup script:

[source,bash]
----
python genai-maf-context-providers/setup.py
----

The script will:

* Load 500 movie plot embeddings into your Neo4j database
* Create the `moviePlots` vector index (1536 dimensions, cosine similarity)
* Create the `moviePlotsFulltext` fulltext index
* Wait for the indexes to come online and verify the setup

You should see `Setup complete!` when it finishes.
