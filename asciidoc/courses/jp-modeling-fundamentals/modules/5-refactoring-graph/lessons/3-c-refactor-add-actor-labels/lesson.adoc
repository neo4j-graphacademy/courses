== クエリのプロファイル

以下のクエリを実行します:

[source,cypher]
----
PROFILE MATCH (p:Person)-[:ACTED_IN]-()
WHERE p.born < '1950'
RETURN p.name
----

このクエリの最初のステップでは、5人のPersonの行が返されることがわかります。

== クエリのプロファイル

グラフをリファクタリングしたので、クエリとプロファイルを再度変更する必要があります。

このクエリを実行する:

[source,cypher]
----
PROFILE MATCH (p:Actor)-[:ACTED_IN]-()
WHERE p.born < '1950'
RETURN p.name
----

このクエリの最初のステップでは、4つのアクター行が返されることがわかります。
