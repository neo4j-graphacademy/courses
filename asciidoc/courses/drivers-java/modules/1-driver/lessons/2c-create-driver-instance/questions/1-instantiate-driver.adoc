.Hint
====
What is the name of the object used to connect a Neo4j instance?
====

[TIP,role=solution]
.Solution
====
The method to create a new connection is `GraphDatabase.driver()`.

[source,Java,role=nocopy noplay]
----
var driver = GraphDatabase.driver(
    NEO4J_URI, 
    AuthTokens.basic(
        NEO4J_USERNAME, 
        NEO4J_PASSWORD)
);

----
