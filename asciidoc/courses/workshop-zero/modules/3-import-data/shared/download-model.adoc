To update the graph data model with the solution, you can:

. Open *Query* and delete all the data from your instance:
+
[source, cypher, role=noplay]
----
MATCH (n) DETACH DELETE n
----
. Download the *{model-name}* data model - link:{models-location}/{model-filename}[{model-filename}^].
. Use the `...` menu, select *Open Model (with data)*, and select the *{model-filename}* file you downloaded.
+
image::../2-import/images/open-model-with-data-annotated.png["... menu and  Open Model (with data) highlighted"]
. Run the import.
