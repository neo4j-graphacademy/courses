== Introduction

Your challenge is to extend the graph data model, importing nodes and relationships from the Northwind dataset:

. Review the source data.
. Identify at least one new node label and one new relationship type.
. Create the node
. Create a relationship that connects the new node to an existing node.
. Run the import
. Write a Cypher query to view the new nodes and relationships.
