Welcome to GraphAcademy, and the Zero to Production workshop!

In this hands-on workshop, you’ll build practical graph skills from the ground up. 

You will:

* Learn the fundamentals of graph databases, Neo4j, and the Cypher query language.
* Model and load relational data into your own Neo4j Aura instance. 
* Explore Neo4j's graph data tools to visualize and display metrics. 
* Create an AI agent on Neo4j Aura—so you can query, explore, and converse with your data.

== Duration

2 hours

== What you will learn

* Basic graph theory
* The structure of a Neo4j graph database
* How to navigate a Neo4j database
* How to write Cypher queries
* Creating a Neo4j Aura instance
* Importing relational data into Neo4j
* Using Aura explore, dashboards, and agents
