== Course Description

In this mini workshop, you will get a conceptual map of **Microsoft Agent Framework (MAF)**: how it relates to Microsoft’s earlier agent stacks, how **agents**, **tools**, and **orchestration** fit together, and how **Model Context Protocol (MCP)** and **agent-to-agent (A2A)** patterns connect to the GraphRAG and Neo4j work from link:/courses/workshop-genai/[Neo4j and Generative AI Workshop^].

The session is **concept-first** (about 30–45 minutes) and does not replace hands-on Neo4j labs.

== Duration

{duration}

== What you will learn

* The purpose and scope of Microsoft Agent Framework
* How agents, tools, and orchestration fit together in MAF-shaped applications
* How Model Context Protocol (MCP) and agent-to-agent (A2A) patterns fit into integration and multi-agent setups
* How Neo4j, GraphRAG, and MCP-related patterns extend what you built in the GenAI workshop

== Prerequisites

To get the most from this mini workshop, you should be comfortable with retrieval, embeddings, and agents at a high level (for example after the GenAI workshop). Completing the following courses first is recommended:

* link:/courses/workshop-genai/[Neo4j and Generative AI Workshop^] - GraphRAG, retrievers, and agents on Neo4j
* link:/courses/genai-fundamentals/[Neo4j & Generative AI Fundamentals^] - Core Generative AI concepts _(optional)_
