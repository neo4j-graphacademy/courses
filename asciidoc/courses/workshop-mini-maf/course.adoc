== Course Description

In this workshop, you will learn about the **Microsoft Agent Framework (MAF)**, including: 

* How MAF relates to Microsoft’s earlier agent stacks
* How **agents**, **tools**, and **orchestration** fit together 
* How **Model Context Protocol (MCP)** and **agent-to-agent (A2A)** patterns connect to the GraphRAG and Neo4j.

The session is **concept-first** (about 30–45 minutes) and does not replace hands-on Neo4j labs.

== Duration

{duration}

== What you will learn

* The purpose and scope of Microsoft Agent Framework
* How agents, tools, and orchestration fit together in MAF-shaped applications
* How Model Context Protocol (MCP) and agent-to-agent (A2A) patterns fit into integration and multi-agent setups
* How Neo4j, GraphRAG, and MCP-related patterns extend what you learned in link:/courses/genai-fundamentals/[Neo4j & Generative AI Fundamentals^] and related GraphAcademy GenAI courses

== Prerequisites

You should have an understanding on Neo4j and generative AI concepts, ideally by completing link:/courses/genai-fundamentals/[Neo4j & Generative AI Fundamentals^]. That course covers embeddings, vector indexes, RAG, GraphRAG, and the Neo4j GraphRAG for Python package; this session assumes you already recognize those ideas.

You can optionally get hands-on knowledge graph construction experience in the GraphAcadmey link:/courses/genai-graphrag-python/[Constructing Knowledge Graphs with Neo4j GraphRAG for Python^] course.
