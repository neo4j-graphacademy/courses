Congratulations on completing the **Mini Workshop: Microsoft Agent Framework (MAF)**!

You now know how to:

* Describe what Microsoft Agent Framework is and how it relates to Semantic Kernel and AutoGen

You now know how to:

* Contrast an agent (policy + model + tools) with a tool (narrow adapter surface)
* Describe orchestration patterns and agent-to-agent (A2A) delegation at a high level
* Relate MCP to a portable contract between model hosts and tools—including Neo4j-oriented MCP material on GraphAcademy
