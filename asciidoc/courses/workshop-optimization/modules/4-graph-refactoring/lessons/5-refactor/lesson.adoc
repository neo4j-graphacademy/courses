.Looking for a scenario?

If you are struggling to identify an scenario or have completed the task above, you can try the following:

Refactor the graph to calculate and store the total cost of an order by summing the price of the products in the order. 

The total cost of an order is calculated from the sum of the line costs of the products in the order. The line cost of a product can be calculated from properties on the `ORDERS` relationship using the formula `LineCost = Quantity * UnitPrice * (1 - Discount)`.

// Solution
// MATCH (order:Order)-[lines:ORDERS]-()
// WITH order, lines.quantity * lines.unitPrice * (1 - lines.discount) as lineCost
// WITH order, sum(lineCost) as totalCost
// SET order.totalCost = totalCost
// RETURN order.orderID, totalCost
=====
