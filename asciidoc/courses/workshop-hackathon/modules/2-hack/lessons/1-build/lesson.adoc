= Hack Time
:type: lesson
:order: 1

The instructed part of this workshop is done.

Now it is time to build something of your own.

== What to build

Continue working in your Codespace repository. Your coding agent has access to the GraphAcademy MCP server — it knows Neo4j, Cypher, graph modeling, and how to structure a GraphRAG application.

Some ideas to get you started:

* Build a GraphRAG chatbot that answers questions about your domain using Neo4j as the knowledge graph
* Create a recommendation engine using collaborative filtering across your graph
* Build an agent that uses `text-to-cypher` to answer natural language questions
* Import a real dataset and explore it with graph queries

Or build something entirely your own.

== Using your coding agent

Your coding agent can use the GraphAcademy MCP server to help you at every step.

[cols="1,3",options="header"]
|===
| Tool | What to use it for

| `answer_question`
| Ask Neo4j or GraphRAG questions — returns grounded answers from GraphAcademy lesson content

| `neo4j_graph_modeler`
| Refine or extend your data model

| `neo4j_query_builder`
| Build and test Cypher queries against your live database schema

| `neo4j_mock_data_generator`
| Generate more test data if you need it

| `neo4j_import_advisor`
| Get help importing a real dataset

| `neo4j_project_builder`
| Scaffold a Neo4j application with driver setup and project structure

| `list_courses`
| Browse GraphAcademy courses for deeper learning on any topic

| `list_lessons`
| List the lessons in a GraphAcademy course

| `list_enrolments`
| View your current GraphAcademy course enrolments

| `enrol`
| Enrol in a GraphAcademy course from within your agent

| `get_next_lesson`
| Fetch the next lesson in a course you are following

| `mark_lesson_complete`
| Mark a lesson as complete and advance through a course

| `verify_lesson`
| Verify your answer or submission for a lesson challenge
|===
