= Time to Build
:order: 2

You have the foundations. Now it is time to build.

In this module, you will use your coding agent — powered by the GraphAcademy MCP server — to build a GraphRAG application against your Neo4j sandbox.
There are no instructions here: you decide what to build, and the tools are there to help you when you need them.

By the end of this module, you will have:

* Built a GraphRAG application connected to your Neo4j sandbox
* Used the GraphAcademy MCP server tools to model, query, and scaffold your application
* Seen first-hand how graph traversal gives an LLM richer, more accurate context
