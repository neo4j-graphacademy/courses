= GraphRAG Hackathon
:categories: workshops, genai
:status: active
:duration: 30 min
:caption: Build a GraphRAG application powered by Neo4j and an AI coding agent
:usecase: blank-sandbox
:key-points: GraphRAG, Neo4j, Cypher, MCP server, AI agents

In this hands-on hackathon, you will learn the fundamentals of GraphRAG and build your own application using your AI coding agent — powered by the GraphAcademy MCP server.

The first 30 minutes cover the essentials: why graphs make LLMs smarter, how to model your domain, and how to write Cypher queries.

Then it's your turn to build. Your coding agent has access to the GraphAcademy MCP server, which provides everything it needs to help you create a working GraphRAG application.

You will learn:

* **Why GraphRAG** — how graphs improve LLM accuracy and reasoning
* **Graph fundamentals** — nodes, relationships, properties, and modeling decisions
* **Cypher** — pattern matching, creating data, indexes, and constraints
* **The GraphAcademy MCP server** — how it powers your coding agent with Neo4j expertise

== Prerequisites

This hackathon is designed for developers. You will need:

* A GitHub account (for GitHub Codespaces)
* Basic programming experience in Python or JavaScript/TypeScript
* An OpenAI API key (or another supported LLM provider)

No prior knowledge of Neo4j or graph databases is required.

== Duration

* **Instructed content:** ~30 minutes
* **Hacking session:** Open-ended
