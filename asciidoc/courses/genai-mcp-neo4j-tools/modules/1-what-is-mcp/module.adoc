= Agentic Systems and MCP
:order: 1

The Model Context Protocol has risen to prominence recently as a way to enable agents to work together in a standardized way.

In this module, you will explore the core concepts of MCP, and how it enables agents to work together. 

You'll learn about the architecture, components, and how to get started with Neo4j's MCP server, including:

* Understand what MCP is and why it matters
* Learn about agents and the ReAct framework
* Explore MCP architecture and components
* Install the Neo4j MCP server and connect it to a Neo4j database
