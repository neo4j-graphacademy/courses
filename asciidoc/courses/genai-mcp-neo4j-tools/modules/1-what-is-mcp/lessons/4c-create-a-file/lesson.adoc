= Install your first MCP server
:type: quiz 
:sequential: true
:order: 4

In the previous lesson, you learned how to configure an MCP server in Claude Desktop. 

In this challenge, you will complete the steps to _install_ the Filesystem server from the MCP SDK example repository.
The server is link:https://www.npmjs.com/package/@modelcontextprotocol/server-filesystem[hosted on NPM^] and can be executed using the `npx` command.

The coordinates of the library are:

----
@modelcontextprotocol/server-filesystem
----

You can link:https://github.com/modelcontextprotocol/servers/tree/main/src/filesystem[view the documentation for the server on Github^].
