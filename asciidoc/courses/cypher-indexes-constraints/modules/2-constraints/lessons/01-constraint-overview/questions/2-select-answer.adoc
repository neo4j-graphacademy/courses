= 2. What type of constraint can be created?

* [x] Key
* [x] Uniqueness
* [x] Existence
* [ ] IS NULL

[TIP,role=hint]
.Hint
====
You cannot create a constraint to ensure a property doesn't exist. 
====

[TIP,role=solution]
.Solution
====
The correct answer is Key, Uniqueness, and Existence.
====
