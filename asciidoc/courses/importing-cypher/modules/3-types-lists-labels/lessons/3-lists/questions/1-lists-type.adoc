[.question]
=  List types?

True or False - values in a list can be of different data types.

- [ ] True
- [*] False

[TIP,role=hint]
.Hint
====
This list would *not* be valid.

`[100, "Banana", 4.1]`
====

[TIP,role=solution]
.Solution
====
The statement is False - All values in a list must have the same data type.
====
