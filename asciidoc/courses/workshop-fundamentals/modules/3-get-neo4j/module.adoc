= Get Neo4j
:order: 3

In this module, you will learn how to get your own Neo4j graph database. You'll explore the various deployment options available, from cloud-hosted solutions to self-hosted instances. Understanding how to set up and access Neo4j is the first step in building graph-powered applications.

You will:

* Learn about the options for running Neo4j
* Create a free Neo4j AuraDB instance in the cloud
* Understand the different deployment models available
