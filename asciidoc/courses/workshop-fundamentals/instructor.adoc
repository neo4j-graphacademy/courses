== Introduction

The Introduction to Graph Databases Workshop is the starting point for anyone new to Neo4j, graph databases, and Cypher.

The workshop introduces the concepts of graph theory, how data is stored and organized in Neo4j, and Cypher queries.

While the concepts covered in this workshop may seem similar and of a beginner level, there are many new concepts introduced and the pace of delivery should reflect this and learners given time to take in the content. 

The workshop can be presented online or in person and is best suited for 90 min to 2 hour sessions.

== Prerequisites

There are no specific learner prerequisites and the workshop should be open to everyone. 

It is assumed the learner has an understanding of how data can be stored in tables.

== Learning objectives

* Graph databases and the underlying principles of Neo4j.
* How to read and write data using Cypher.
* Patterns matching and how to use it to find data in a graph.
* How to aggregate data and asking questions of your data.

== Outline plan

(All timings are estimates and adaptable)

=== Introduction

(10 mins)

*Introduce yourself, the session, and have participants login*

Use the introduction slides to introduce yourself and the agenda

Introduce GraphAcademy and ask learners to create an account and goto the workshop URL in the notes document.

https://graphacademy.neo4j.com/courses/workshop-fundamentals/

Starter activity - Ask attendees “What do you know about graph databases?”

Show learners the GraphAcademy course summary, summarise the learning objectives and enrol in the course.

=== Graph Databases

(30-45 mins)

*Learn core concepts and how to store data in Neo4j*

Describe core Neo4j principles relating to Nodes, Labels, Properties, and Relationships.

Position Neo4j as different to other database technologies.

Spend time describing Cypher patterns, these are core to using Neo4j.

The module gets learners to use Cypher statements and observe outcomes before going into the detail of how they are constructed. The pattern of Predict and Run first is intentional to build understanding without overwhelming the learner with detail. Let participants explore the Cypher statements before going into detail.

=== Cypher

(45-60 mins)

*Explore the Cypher query language structure and features*

The module explores commonly used Cypher language topics - matching, filtering, ordering, limits, aggregation, WITH, shortest paths.

Each lesson follows a similar structure of - describe the concepts, run and predict the outcome of prebuilt Cypher queries, complete challenges.

For each challenge the learner has to “add the missing parts” and replace the question marks. Solutions are given and learners should be encourage to look only once they have attempted the challenge.

Learners should be given time to complete the challenges, this is where the majority of learning will take place.

=== Close

Return to the introduction slides, remind learners they will have access to the workshop content indefinitely.

Run a Q&A session.

Suggest GraphAcademy courses for learners to continue their learning.
