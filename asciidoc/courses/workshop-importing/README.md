# Northwind Import Workshop

A comprehensive 2-hour workshop teaching graph data modeling and importing the Northwind dataset into Neo4j Aura.

## Workshop Structure

### Module 1: Getting Started with Neo4j Aura (3 lessons)
1. What is Neo4j Aura?
2. Create Your Aura Instance
3. Aura Tools Overview

### Module 2: Graph Data Modeling Fundamentals (7 lessons)
1. What is Graph Data Modeling?
2. Understanding the Northwind Domain
3. Modeling Nodes and Labels
4. Modeling Relationships
5. Defining Properties
6. Testing Your Data Model
7. Refactoring Your Graph Model

### Module 3: Importing the Northwind Dataset (6 lessons)
1. Data Import Overview
2. Introduction to Data Importer
3. Importing Northwind Nodes
4. Creating Relationships
5. Constraints and Indexes
6. Running the Import

### Module 4: Querying the Northwind Graph (5 lessons)
1. Introduction to Cypher
2. Reading Data from Northwind
3. Advanced Pattern Matching
4. Filtering Data with WHERE
5. Aggregating and Analyzing Data

## Total: 21 Lessons

## Key Features

- All graph diagram images replaced with Mermaid diagrams
- Adapted content from modeling-fundamentals, importing-fundamentals, and neo4j-fundamentals
- Northwind dataset as the practical example throughout
- Hands-on exercises with real business data
- Complete from setup to advanced queries

## Northwind Dataset

The workshop uses the Northwind dataset, representing a food products company with:
- 91 Customers
- 830 Orders  
- 77 Products
- 8 Categories
- 29 Suppliers
- 9 Employees
- ~4,000 relationships

## Learning Outcomes

By completing this workshop, participants will:
- Set up and use Neo4j Aura
- Understand graph data modeling principles
- Import CSV data into Neo4j
- Write Cypher queries to explore and analyze data
- Apply best practices for constraints and indexes
