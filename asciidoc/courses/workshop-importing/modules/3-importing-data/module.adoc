= Importing the Northwind Dataset
:order: 3

In this module, you will import the Northwind dataset into your Neo4j Aura instance using the Data Importer tool.

You'll transform CSV data into a graph database by creating nodes, relationships, and properties based on the data model you learned in the previous module.

In this module, you will:

* Learn about data import options in Neo4j
* Use the Data Importer tool to load CSV files
* Create nodes for all Northwind entities
* Create relationships between nodes
* Add constraints and indexes for performance
* Complete the full Northwind graph import
