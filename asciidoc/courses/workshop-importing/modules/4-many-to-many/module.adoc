= Many-to-Many Relationships
:order: 4

In this module, you will discover why graphs excel at many-to-many relationships.

By the end of the module, you will:

* Learn how to transform pivot tables into direct graph relationships
* Create IN_CATEGORY relationships between Product and Category nodes
* Create CONTAINS relationships between Order and Product nodes with properties
* Build the complete Customer→Order→Product path that enables recommendations
* Write multi-hop traversal queries across the entire graph
