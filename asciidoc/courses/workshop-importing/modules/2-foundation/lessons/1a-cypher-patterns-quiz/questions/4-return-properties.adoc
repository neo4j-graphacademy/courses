[.question.select-in-source]
= Accessing Node Properties

Complete the query to return the product name property.

[source,cypher,role=nocopy noplay]
.Return the product name
----
MATCH (p:Product)
WHERE p.unitPrice > 50
RETURN p/*select:.productName*/;
----

- [ ] ->productName
- [x] .productName
- [ ] [productName]
- [ ] :productName

[TIP,role=hint]
.Hint
====
Properties are accessed using dot notation, similar to many programming languages.
====

[TIP,role=solution]
.Solution
====
The correct syntax to access a property is `.productName` (dot notation).

Properties use dot notation: `variable.propertyName`

[source,cypher,role=nocopy noplay]
----
/* Find expensive products */
MATCH (p:Product)
WHERE p.unitPrice > 50
RETURN p.productName;
----
====
