[.verify.slide]
= Validate Import

Once you have imported the Category nodes, click the **Check Database** button to verify that the task is complete.

verify::[]

[TIP,role=hint]
.Hint
====
Follow the same steps as the previous lessons where you imported other node types.

Make sure to:

* Set the label to `Category`
* Map from `categories.csv`
* Set `categoryId` as the ID property
* Click Preview to review
* Click Run Import
====

[TIP,role=solution]
.Solution
====
The Category node should be configured with:

* **Label:** `Category`
* **File:** categories.csv
* **ID property:** categoryId
* **Key properties:** categoryName, description

After running the import, you should have 8 Category nodes in your graph.
====
