= Importing data into Neo4j
:order: 1

In this module, you will explore the different ways to import data into Neo4j and learn about the various tools and techniques available for importing data. You'll discover how to choose the right import method for your use case, whether you're working with CSV files, APIs, or other data sources. Understanding these import options is crucial for efficiently populating your Neo4j database with the data you need to build powerful graph applications.

In this module, you will:

* Explore the different ways to import data into Neo4j
* Learn about the different tools and techniques available to import data
* Understand when to use each import method
