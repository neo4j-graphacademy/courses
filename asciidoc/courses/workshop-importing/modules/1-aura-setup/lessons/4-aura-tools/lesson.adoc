== Exploring Aura Tools

You created your Aura instance and saved your credentials. To work with your database, you need to understand the tools available in Neo4j Aura.


In this lesson, you will learn about the key tools in Neo4j Aura.


[.slide]
== Understanding the Aura Console

Neo4j Aura provides four key tools to work with your database:


* **Data Importer** - Import CSV files
* **Query Tool** - Write Cypher queries
* **Explore** - Visualize graph data
* **Instance Management** - Backups, settings, and security


[.slide.col-2]
== Using the Data Importer

[.col]
====
The **Data Importer** is a visual tool for importing CSV data into Neo4j.


**What you can do:**

* Upload CSV files
* Design your graph data model visually
* Map CSV columns to nodes and relationships
* Set up constraints and indexes
* Run the import with one click
====

== Writing queries with the Query tool

The **Query** tool (also called Neo4j Browser or Query Console) lets you:


* Write and execute Cypher queries
* Visualize query results as graphs or tables
* Explore your data interactively
* Save and share queries


[.slide]
== Visualizing data with the Explore tool

The **Explore** tool provides:


* Interactive graph visualization
* Point-and-click navigation
* Node and relationship inspection
* Quick insights into your data


This tool lets you explore your data without writing queries.


[.slide]
== Managing your instance

From the instance details page, you can:


* View connection information
* Create snapshots (backups)
* Monitor database metrics
* Manage users and security
* Configure instance settings


[.slide]
== Accessing the Tools

To access these tools from the Aura console:


1. Click on your instance
2. Click **Open** or click the arrow icon
3. Choose the tool you want to use
