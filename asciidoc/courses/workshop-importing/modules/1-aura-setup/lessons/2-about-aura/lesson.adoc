== Introduction

To build your graph, you need a Neo4j database. This lesson introduces Neo4j Aura, the fully managed cloud database service you'll use for the workshop.


In this lesson, you will learn what Neo4j Aura is, the different tiers available, and why the Free tier is perfect for this workshop.


[.slide]
== What is Neo4j Aura?

Neo4j Aura is a fully managed database-as-a-service (DBaaS) for Neo4j graph databases.


**Fully managed means:**

* **No infrastructure to manage** - Neo4j handles servers, maintenance, and operations
* **Built-in reliability** - 99.95% uptime SLA with fault tolerance
* **Automatic backups** - Your data is protected with regular automated backups
* **Always up to date** - Software updates and security patches are applied automatically
* **Enterprise security** - Encryption at rest and in transit, plus network isolation


You focus on building your graph and querying data. Aura handles everything else.


[.slide]
== Aura vs. Self-Managed Neo4j

The key difference between Aura and self-managed Neo4j is who handles operational responsibilities.


**With self-managed Neo4j:**

* You install and configure servers
* You manage backups and disaster recovery
* You apply updates and security patches
* You monitor performance and scale infrastructure
* You handle high availability setup


**With Aura:**

* Neo4j manages all infrastructure
* Automated backups with point-in-time recovery
* Updates applied with zero downtime
* Built-in monitoring and alerting
* High availability included (on higher tiers)


For this workshop, Aura lets you focus on learning graph modeling and Cypher queries without managing database infrastructure.

== Aura Tiers

Neo4j Aura offers four tiers designed for different project stages and business needs:

=== Aura Free

**Designed for learning and experimentation**


**Limits:**

* Up to 200,000 nodes
* Up to 400,000 relationships
* Single instance (no high availability)
* On-demand hourly backups


**Best for:**

* Learning Neo4j
* Building proofs of concept
* Workshop environments
* Development and testing

[.slide.discrete]
=== Aura Professional

**Designed for production applications with moderate requirements**


**Features:**

* More storage space
* Choice of cloud provider (AWS, GCP, Azure)
* Daily automated backups
* Single-instance deployment


**Best for:**

* Production applications
* Applications with predictable workloads
* Teams needing cloud provider flexibility


**Cost:** Pay-as-you-go based on usage


[.slide.discrete]
=== Aura Business Critical

**Designed for enterprise applications requiring high availability**


**Features:**

* Guaranteed uptime with multi-AZ clusters
* Role-based access control (RBAC)
* Single sign-on (SSO)
* Hourly backups
* High availability architecture


**Best for:**

* Mission-critical applications
* Enterprise-scale deployments
* Applications requiring maximum uptime


**Cost:** Premium pricing for enterprise features


[.slide.discrete]
=== Aura Virtual Dedicated Cloud

**Designed for strict compliance and security requirements**


**Features:**

* Dedicated infrastructure
* Customer-managed encryption keys (CMEK)
* VPC isolation
* Highest security and compliance


**Best for:**

* Applications with regulatory requirements
* Financial services and healthcare
* Organizations requiring dedicated resources


**Cost:** Custom pricing for dedicated infrastructure

The Aura Free tier is perfect for this workshop because:

== Built-in Tools

Aura includes built-in tools accessible through your web browser:


**Data Importer**

* Visual CSV import tool
* Map CSV columns to nodes and relationships
* No coding required


**Query tool (Neo4j Browser)**

* Write and execute Cypher queries
* Visualize query results as graphs
* Explore data interactively


**Explore tool**

* Point-and-click graph exploration
* Inspect nodes and relationships
* Quick data insights without queries


**Instance Management**

* Monitor database metrics
* Create backups and snapshots
* Manage users and security


You'll use these tools throughout the workshop.

== Getting Your Free Instance

Creating an Aura Free instance takes just a few minutes:


1. Sign up at neo4j.com/cloud/aura (using email, Google, or GitHub)
2. Create a new Free instance
3. Save your credentials (username and password)
4. Start building your graph
