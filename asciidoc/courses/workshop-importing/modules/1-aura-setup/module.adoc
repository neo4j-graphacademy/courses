= Getting Started with Neo4j Aura
:order: 1

In this module, you will learn how to set up a free Neo4j Aura instance to host your Northwind database.

Neo4j Aura is a fully managed cloud database service that handles all the infrastructure, backups, and updates for you.

In this module, you will:

* Learn what Neo4j Aura is and its benefits
* Create a free Aura database instance
* Explore the tools available in the Aura console

If you are ready, let's get going!
