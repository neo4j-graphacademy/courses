== Loading the CSV files

Load all the Northwind CSV files into your Data Importer:

button::Load Northwind Data Files[role=NX_LOAD_DATA_FILES]

This will load all the files you'll need for the workshop:

* **products.csv** - 77 products (you'll import these now)
* **customers.csv** - 91 customers (optional import in next lesson)
* **orders.csv** - 830 orders (optional import later)
* **order_details.csv** - 2,155 order items (for CONTAINS relationships in Module 4)
* **categories.csv** - 8 categories (optional enrichment in Module 7)

**You're ready to import your first nodes!**

[.slide]
== Creating a product node

1. Click **Add Node**
2. Set the label to `Product`
3. Click **Map from file**
4. Select `products.csv`
5. Review and configure property mappings
6. Set `productId` as the ID property (key icon)
7. Click **Run Import**

[.slide]
== Validating your import

[source,cypher]
.Count all products
----
MATCH (p:Product)
RETURN count(p) AS productCount;
----

You should see **77 products**.

[source,cypher]
.View first 5 products
----
MATCH (p:Product)
RETURN p.productName, p.unitPrice, p.unitsInStock
LIMIT 5;
----

**✅ Success!** You have imported your first nodes.
