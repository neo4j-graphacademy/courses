In this hands-on workshop, you will learn how to import a real-world dataset into Neo4j Aura.

The Northwind dataset represents a fictitious company that sells food products. It contains data about customers, orders, products, suppliers, employees, and more.

You will learn:

* How to set up a free Neo4j Aura instance
* Graph data modeling fundamentals
* How to design a graph model for the Northwind domain
* How to import CSV data into Neo4j
* How to write Cypher queries to explore and analyze your data

By the end of this workshop, you will have a fully populated Northwind graph database and the skills to import your own data into Neo4j.

== Prerequisites

This workshop is designed for beginners. No prior knowledge of Neo4j or graph databases is required.

Basic familiarity with databases and SQL concepts will be helpful but not essential.

== Duration

2 hours

== What you will learn

* The benefits of Neo4j Aura and how to create a free instance
* The fundamentals of graph data modeling
* How to design nodes, labels, relationships, and properties
* Techniques for importing CSV data into Neo4j
* How to write Cypher queries to explore and analyze graph data
* Best practices for data modeling and importing
