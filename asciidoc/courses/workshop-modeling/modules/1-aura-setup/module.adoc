= Getting Started with Neo4j Aura
:order: 1

In this module, you will set up your workshop environment by creating a free Neo4j Aura instance.

First, you'll get an overview of what you'll accomplish in this workshop and the benefits of graph databases for connected data.

Then you'll learn about Neo4j Aura and create your free database instance.

In this module, you will:

* Understand the workshop goals and what you'll build
* Learn about Neo4j Aura and the different tiers available
* Create your free Aura database instance
* Explore the built-in tools in the Aura console

If you are ready, let's get going!
