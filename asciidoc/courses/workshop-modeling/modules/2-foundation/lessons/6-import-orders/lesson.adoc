== Importing Order Nodes

You imported Product and Customer nodes. Now import Order nodes using the same workflow.

In this challenge, you will work independently to import Order nodes from the orders.csv file.

[.slide]
== File structure: orders.csv

The `orders.csv` file contains order information:

[options="header"]
|===
| Column | Type | Description
| orderID | Integer | Unique identifier for each order
| customerID | String | Reference to customer who placed the order
| orderDate | DateTime | Date the order was placed
| shipName | String | Name for shipping
| shipAddress | String | Shipping street address
| shipCity | String | Shipping city
| shipCountry | String | Shipping country
|===

This file has **830 rows** - one for each order placed.

== Map the Order node

In the **Data Modeling tab**:

. Select the **Define manually** option. This will open the data modeling canvas with a new node.
. **Set the label:** With the new node selected, under **Definition**, set the label to `Order`
. **Select the table:** Under **Table**, select `orders.csv`


[.slide]
== Configure properties

Use the **Map from table** button to select the following columns. Verify their types and rename as applicable:

[options="header"]
|===
| Column | Type | Rename To
| orderID | Integer | id
| orderDate | DateTime | date
| shipCountry | String |
| shipCity | String |
|===

Verify the ID property is set to `id` and update if necessary.


[.slide]
== Date Properties

The `orderDate` column should be converted to a `DateTime` type.

The Import tool will coerce the string value into a `DateTime` object during import.

To run comparisons on this property in Cypher, you must use the `datetime()` function to convert string values:

[source,cypher]
----
MATCH (o:Order)
WHERE o.date > datetime('2024-01-01')
RETURN o
----


[.slide]
== Run the import

Click the **Run Import** button to import the data. You will see a progress indicator while the import is running.

Once the import is complete, you will see a summary of the import, including the Cypher statements that were executed during the import process.

You should see confirmation that **830 Order nodes** were created.
