== Design Your Graph Model

Before importing data, let's design your graph data model. You can use AI to help identify nodes, relationships, and properties from your data.

**How this works:**

1. Describe your data or problem in natural language
2. AI will help identify graph elements (nodes, relationships, properties)
3. You'll refine the model before importing

[.slide]
== What AI can help you identify

Tell me about the problem you are trying to solve, or the data you're working with. You can:

* Describe your data in natural language
* Paste a few lines from a CSV file
* Describe your database schema

I can help you to:

* Identify **nodes** - Extract the nouns (things) from your data
* Identify **relationships** - Find the verbs (connections) between nodes
* Identify **properties** - Determine attributes that describe nodes and relationships
* Suggest **labels** - Name your node types appropriately

[.slide]
== Example: Northwind dataset

For the Northwind dataset we're using in this workshop, the AI would identify:

**Nodes:** Product, Customer, Order, Category

**Relationships:** 
* Customer PLACED Order
* Order CONTAINS Product
* Product IN_CATEGORY Category

**Properties:**
* Product: name, price, unitsInStock
* Customer: companyName, contactName, country
* Order: date, shipCountry

== After modeling

Once you've designed your model:

1. **Download data files** - Get CSVs or connect to your database
2. **Load into Import tool** - Upload all files at once
3. **Create nodes** - Map files to node labels
4. **Create relationships** - Connect nodes based on your model
5. **Query your graph** - Test with Cypher queries

In the next lesson, you'll use the Import tool to bring your model to life.
