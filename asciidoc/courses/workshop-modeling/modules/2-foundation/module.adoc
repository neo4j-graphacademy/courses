= Building Your Graph
:order: 2

In this module, you will learn graph fundamentals, understand how to use the Import tool, and import your first nodes.

By the end of the module, you will:

* Understand the building blocks of a graph (nodes, relationships, properties, labels)
* Learn enough Cypher to query your data
* Know how to identify what should be a node in your graph model
* Learn how to use the Import tool to map data to your graph model
* Import Product nodes into your graph - the foundation of your recommendation system
