[.question]
= Node Identification

Which of the following should be a **node** in a graph model?

- [ ] A. Product price (unitPrice)
- [ ] B. Order date (orderDate)
- [x] C. Customer
- [ ] D. Surname

[TIP,role=hint]
.Hint
====
Nodes represent entities - distinct things with identity, properties, and connections.
====

[TIP,role=solution]
.Solution
====
The correct answer is **Customer**.

Product price (unitPrice), Order date (orderDate) are properties that describe other entities, not entities themselves.
Surname is a property that describes a customer, not an entity itself.
====
