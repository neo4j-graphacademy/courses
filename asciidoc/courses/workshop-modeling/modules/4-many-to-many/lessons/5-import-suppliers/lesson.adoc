== Importing Suppliers

You have Product nodes. Now import Supplier nodes and create SUPPLIES relationships to connect them.

In this challenge, you will work independently to import suppliers and create relationships between suppliers and products.

[.slide]
== File structure: suppliers.csv

The link:https://data.neo4j.com/northwind/suppliers.csv[suppliers.csv^] file contains supplier information (click the filename to download if needed):

[options="header"]
|===
| Column | Type | Description
| supplierID | Integer | Unique identifier for each supplier
| companyName | String | Supplier company name
| contactName | String | Contact person at the supplier
| contactTitle | String | Title of the contact person
| address | String | Street address
| city | String | City location
| region | String | Region/state (may be empty)
| postalCode | String | Postal/ZIP code
| country | String | Country location
| phone | String | Phone number
| fax | String | Fax number (may be empty)
| homePage | String | Website URL (may be empty)
|===

This file has **29 rows** - one for each supplier.


[.slide]
== Understanding SUPPLIES relationships

**Business rule:** Each supplier provides one or more products.

The `supplierID` column in products.csv references suppliers - you'll use this to create SUPPLIES relationships.

**Graph pattern:**

[source,mermaid]
----
graph LR
    Supplier((Supplier<br/>supplierID: 1)) -->|SUPPLIES| Product((Product<br/>productName: 'Chai'))

    style Supplier fill:#B983FF
    style Product fill:#FCC644
----


[.slide]
== Your challenge

Using what you've learned about the Import tool:

1. **Create a Supplier node mapping**
   * Map suppliers.csv to a Supplier node
   * Set `supplierID` as the unique identifier
   * Include relevant properties (companyName, contactName, city, country, phone)

2. **Create SUPPLIES relationships**
   * Map products.csv as the relationship source
   * Create relationship type: `SUPPLIES`
   * **From:** Map `supplierID` in products.csv to Supplier node's ID
   * **To:** Map `productID` in products.csv to Product node's ID

3. **Run the import**


[.slide]
== Expected results

After importing, you should have:

* **29 Supplier nodes** created
* **77 SUPPLIES relationships** created (one for each product)

You can verify with this query:

[source,cypher]
----
MATCH (s:Supplier)-[:SUPPLIES]->(p:Product)
RETURN s.companyName AS supplier,
       count(p) AS productsSupplied
ORDER BY productsSupplied DESC
LIMIT 5;
----
