== Validate Results

Once you have imported the Supplier nodes and created the SUPPLIES relationships, verify your results.

verify::[]

[TIP,role=hint]
.Hint
====
Make sure you completed both steps:

**Check:**

* Supplier node mapping created with supplierID as unique identifier
* SUPPLIES relationship created using products.csv as source
* From: supplierID (in products.csv) → Supplier node
* To: productID (in products.csv) → Product node
* Import executed successfully
====

[TIP,role=solution]
.Solution
====
Run this query to check your Supplier nodes and SUPPLIES relationships:

[source,cypher]
.Check Suppliers and relationships
----
MATCH (s:Supplier)
WITH count(s) AS supplierCount
MATCH ()-[r:SUPPLIES]->()
RETURN supplierCount, count(r) AS suppliesCount;
----

You should see 29 suppliers and 77 SUPPLIES relationships.

**If verification fails:**

* Verify the suppliers.csv file was uploaded
* Check that Supplier node has supplierID as unique identifier
* Verify SUPPLIES relationship uses products.csv (not suppliers.csv)
* Ensure From field maps supplierID → Supplier, To field maps productID → Product
* Try reloading the Import tool page and checking your mappings
