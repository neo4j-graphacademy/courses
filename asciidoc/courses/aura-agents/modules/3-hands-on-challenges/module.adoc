= Build and Publish Agents
:order: 3

== Module Overview

In this module, you will build your own custom agent and publish it so it can be called from external applications.

You will:

* Build an agent using Northwind or your own AuraDB instance
* Publish your agent with External access and MCP server enabled
* Connect to your agent from Claude Desktop or Cursor
* Understand when and how to use MCP vs REST API
