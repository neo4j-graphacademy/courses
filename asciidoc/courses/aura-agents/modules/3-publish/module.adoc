== Module Overview

In this module, you will publish your agent so it can be called from external applications like Cursor.

By the end of this module, you will be able to:

* Compare Internal and External access and choose the appropriate mode for your use case
* Enable the MCP server
* Connect your agent to Cursor using the MCP endpoint and test it with real prompts
