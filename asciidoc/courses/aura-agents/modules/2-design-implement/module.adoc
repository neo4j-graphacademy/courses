= Building Agents
:order: 2

== Module Overview

In this module, you will design and implement a Northwind Analyst agent with Cypher Template and Text2Cypher tools, then complete a hands-on challenge to build it end to end.

You will learn:

* How to design an agent: role, scope, and effective tool descriptions
* How to create an agent from scratch or generate a starting point with the Create with AI option
* How to implement Cypher Template tools with Northwind Cypher queries and parameters
* How to add a Text2Cypher fallback tool and when to use it instead of templates
