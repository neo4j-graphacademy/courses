= Building an Agent
:order: 2

== Module Overview

In this module, you will design a Northwind Analyst agent, build it two different ways, configure it, then extend it by adding new tools.

You will learn how to:

* Design an agent: define role, scope, question types, and effective tool descriptions
* Use Create with AI to generate a working agent from a prompt, investigate the results, and trace through a query
* Edit, add, delete tools, and delete an agent
* Build an agent from scratch with all three available tool types
* Add new tools to extend an existing agent
