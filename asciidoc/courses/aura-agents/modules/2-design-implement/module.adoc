== Module Overview

In this module, you will set up Northwind, design an agent, and build it with Cypher Template and Text2Cypher tools. You will complete a hands-on challenge to configure and test your agent.

You will learn:

* How to set up Northwind and enable tool authentication
* How to write effective prompts and configure agent behavior
* How to implement agents with Cypher Template and Text2Cypher tools
