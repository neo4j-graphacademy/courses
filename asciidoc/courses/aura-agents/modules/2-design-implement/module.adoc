== Module Overview

In this module, you will learn how to design and implement Aura Agents using Cypher and tools.

* How to write effective prompts and configure agent behavior
* How to implement agents with Cypher Template, Similarity Search, and Text2Cypher tools
* How to leverage GraphRAG for advanced retrieval
