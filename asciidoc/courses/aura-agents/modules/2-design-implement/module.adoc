== Module Overview

In this module, you will inspect what the AI has created, create an agent from scratch using your own knowledge graph, then customise your agents by adding your own tools.

You will:

* Inspect and explain the tools the AI created and when each is used
* Customise to add your own tools: edit, add, delete tools, build from scratch, and extend with new tools
