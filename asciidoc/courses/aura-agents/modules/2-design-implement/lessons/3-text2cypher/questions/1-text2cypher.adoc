[.question]
= Choosing the Right Tool

For a question like "Which customers ordered products from more than two different suppliers?", which tool should the agent use?

* [ ] Cypher Template
* [x] Text2Cypher
* [ ] Similarity Search
* [ ] All of the above

[TIP,role=hint]
.Hint
====
Text2Cypher is for questions where the query structure itself changes: different MATCH patterns, dynamic filters, or aggregations you cannot define in advance.
====

[TIP,role=solution]
.Solution
====
**Text2Cypher.**

The filter count, the multi-hop pattern, and what to return all depend on the question. You cannot write this query in advance with only parameter slots. Text2Cypher generates the Cypher at runtime.
====
