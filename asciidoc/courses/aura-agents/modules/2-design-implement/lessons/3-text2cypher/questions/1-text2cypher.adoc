[.question]
= Name and Description for Text2Cypher

In the agent configuration, why does a Text2Cypher tool need both a name and a description?

Select the correct answer.

* [ ] They are optional if you already have Cypher Template tools
* [x] The agent uses the name to identify the tool and the description to decide when to use it
* [ ] They replace the automatically supplied database schema
* [ ] They are only shown in the Aura Console and do not affect tool selection

[TIP,role=hint]
.Hint
====
The lesson states what each field is for: one identifies the tool, the other guides when the LLM should choose it.
====

[TIP,role=solution]
.Solution
====
**The agent uses the name to identify the tool and the description to decide when to use it.**

You must give the tool a name and a description. The name identifies the tool in the configuration and reasoning output. The description is what the model uses to decide whether this tool fits the user's question relative to your other tools. The database schema is supplied automatically. The description adds when-to-use and domain context.
====
