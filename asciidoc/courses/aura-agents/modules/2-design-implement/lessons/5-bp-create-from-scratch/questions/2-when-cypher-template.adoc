[.question]
= When to Use a Cypher Template

When should you use a Cypher Template tool instead of Text2Cypher?

Select the correct answer.

* [ ] When the query structure changes depending on the user's question
* [x] When the complete Cypher query structure is fixed and only parameter values change
* [ ] When the LLM cannot access the graph schema
* [ ] When the agent needs to answer questions outside the graph

[TIP,role=hint]
.Hint
====
Apply the lesson test: can you write the full Cypher now with only `$parameter` slots for values? If yes, prefer a template for reliability.
====

[TIP,role=solution]
.Solution
====
**When the complete Cypher query structure is fixed and only parameter values change.**

That is when a Cypher Template fits: same MATCH, WHERE, and RETURN shape every time. The LLM only supplies values such as IDs or limits. When the pattern or filters themselves change with the question, Text2Cypher is the usual choice. The schema is available to Text2Cypher automatically. Agent tools query the graph. They are not for arbitrary off-graph answers.
====
