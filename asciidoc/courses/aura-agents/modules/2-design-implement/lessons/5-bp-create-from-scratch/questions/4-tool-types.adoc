[.question]
= Tool Types Available in Aura Agent

What types of tools can an Aura Agent use?

* [x] Cypher Template: predefined parameterized Cypher queries
* [x] Text2Cypher: natural language to Cypher at runtime
* [x] Similarity Search: when the graph has a vector index
* [ ] Stored procedures: call Neo4j procedures from the agent
* [ ] API tools: call external REST APIs from the agent

[TIP,role=hint]
.Hint
====
Aura Agent tools are read-only and run against your graph. Think about which options are actually available in the agent configuration (Add Tool menu).
====

[TIP,role=solution]
.Solution
====
**Cypher Template, Text2Cypher, and Similarity Search are correct.**

These are the three tool types you can add in the Aura Agent configuration. Cypher Template runs pre-written parameterized Cypher. Text2Cypher generates Cypher from natural language at runtime. Similarity Search is available when your graph has vector embeddings and a vector index.
====
