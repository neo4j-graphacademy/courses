[.question]
= Tools You Can Add

Which of these can you add to an Aura Agent in the Console?

Select all that apply.

* [x] Cypher Template: run predefined parameterized Cypher
* [x] Text2Cypher: generate Cypher from natural language at runtime
* [x] Similarity Search: find nodes by vector similarity (when you have a vector index)
* [ ] Call a REST API or external service from the agent
* [ ] Run a Neo4j stored procedure that writes to the graph

[TIP,role=hint]
.Hint
====
Aura Agent tools are read-only and query your graph. Check the Add Tool menu: which of these options are actually there?
====

[TIP,role=solution]
.Solution
====
**Cypher Template, Text2Cypher, and Similarity Search.**

Those are the three tool types in the Aura Agent configuration. All are read-only and run against your graph. The agent cannot call arbitrary REST APIs or run stored procedures that write data. Those would be different integration patterns, not the built-in agent tools.
====
