[.question]
= Applying Tool Changes

After adding a new Cypher Template tool to an agent, what must you do for the change to take effect?

* [ ] Close and reopen the agent configuration
* [ ] Test the agent with a sample question
* [x] Click **Update agent** to persist the change
* [ ] Delete and recreate the agent

[TIP,role=hint]
.Hint
====
The agent continues to use the previous configuration until you explicitly apply your changes.
====

[TIP,role=solution]
.Solution
====
Click **Update agent** to persist the change.

Edits, additions, and deletions to tools are not applied until you click **Update agent**. The agent keeps using the previous configuration until then.

**Why the others are wrong:**

* Close and reopen: This does not apply changes; it only refreshes the view.
* Test with a sample question: Testing verifies the change but does not save it. You must click Update agent first.
* Delete and recreate: Unnecessary. Use Update agent to save your changes.
====
