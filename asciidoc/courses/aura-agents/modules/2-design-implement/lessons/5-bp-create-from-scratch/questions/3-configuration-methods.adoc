= Applying Tool Changes

You add a new Cypher Template to your agent and run a test question. The agent still behaves as before and does not use the new tool. What is missing?

Select the correct answer.

* [ ] Restart the browser or reconnect to the instance
* [ ] Run a question that explicitly mentions the new tool
* [x] Click **Update agent** to save the configuration. Changes are not applied until you do
* [ ] Clear the agent cache in project settings

[TIP,role=hint]
.Hint
====
The agent keeps using the last saved configuration until you explicitly persist your edits.
====

[TIP,role=solution]
.Solution
====
**Click Update agent to save the configuration.**

Adding, editing, or removing tools is only applied when you click **Update agent**. Until then the agent keeps using the previous configuration. Restarting the browser or reconnecting does not save changes. Testing does not persist them either.
====
