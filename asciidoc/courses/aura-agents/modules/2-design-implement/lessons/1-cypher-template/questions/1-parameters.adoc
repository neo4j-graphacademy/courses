[.question]
= Cypher Template Parameters

When should you use a Cypher Template instead of Text2Cypher?

* [ ] When the query structure changes between questions
* [x] When the MATCH pattern, WHERE conditions, and RETURN clause are fixed and only values change
* [ ] When you need semantic similarity matching
* [ ] When the user's question is unpredictable

[TIP,role=hint]
.Hint
====
Cypher Template is for common, repeated questions with known structure. The LLM extracts parameter values and runs your predefined query.
====

[TIP,role=solution]
.Solution
====
**When the MATCH pattern, WHERE conditions, and RETURN clause are fixed and only values change.**

Use a Cypher Template when you can write the complete Cypher right now with only `$parameter` slots for variable values. The query structure is deterministic.
====
