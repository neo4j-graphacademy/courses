= When to Use Cypher Template

When should you use a Cypher Template instead of Text2Cypher?

Select the correct answer.

* [ ] When the query structure changes from one question to the next
* [x] When the query shape is fixed and only values (e.g. IDs, limits) change
* [ ] When you need similarity search over embeddings
* [ ] When the user's question is unpredictable

[TIP,role=hint]
.Hint
====
Cypher Template runs one predefined query with parameter slots. If you can write the full Cypher now and only plug in values, a template fits.
====

[TIP,role=solution]
.Solution
====
**When the query shape is fixed and only values change.**

Use a Cypher Template when the MATCH pattern, WHERE conditions, and RETURN are the same every time and only inputs like customer ID or limit change. The LLM extracts those values and runs your query. If the pattern or filters change with the question, use Text2Cypher instead.
====
