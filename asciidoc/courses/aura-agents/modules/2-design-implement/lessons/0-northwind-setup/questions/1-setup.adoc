= Northwind Setup

What must you enable so your AuraDB instance appears when creating an agent?

* [ ] Vector-optimized configuration
* [ ] Similarity Search
* [x] Tool authentication
* [ ] Generative AI Assistance only

[TIP,role=hint]
.Hint
====
Agents need permission to connect to and query your AuraDB instance. This is configured in Security Settings.
====

[TIP,role=solution]
.Solution
====
**Tool authentication.**

Enable "Allow tools to connect with permissions from the user's project role" in Aura Console → Organization → Security Settings, then select your Northwind instance. Without it, your instance will not appear in the instance dropdown when creating an agent. Also verify Generative AI Assistance and Aura Agent are enabled.
====
