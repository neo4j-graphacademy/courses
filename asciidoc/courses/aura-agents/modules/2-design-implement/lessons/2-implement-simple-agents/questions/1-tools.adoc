= Agent Tools

What types of tools can an Aura Agent use?

* [ ] Only Cypher queries
* [ ] Only external APIs
* [x] Cypher procedures, stored procedures, and API tools
* [ ] Only pre-built Neo4j functions

[TIP,role=hint]
.Hint
====
Agents invoke multiple tool types. Cypher procedures run queries, stored procedures run custom logic, and API tools call external endpoints.
====

[TIP,role=solution]
.Solution
====
**Cypher procedures, stored procedures, and API tools.**

Aura Agents support multiple tool types: Cypher procedures for graph queries, stored procedures for custom Neo4j logic, and API tools for external services.
====
