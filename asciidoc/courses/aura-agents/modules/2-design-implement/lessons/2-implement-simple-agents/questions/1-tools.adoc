[.question]
= Aura Agent Tool Types

What tool types can an Aura Agent use? Select all that apply.

* [x] Cypher Template
* [x] Text2Cypher
* [x] Similarity Search
* [ ] External HTTP API calls
* [ ] Stored procedures
* [ ] Custom Python functions

[TIP,role=hint]
.Hint
====
Aura Agent supports exactly three read-only tool types. One runs predefined Cypher with parameters, one generates Cypher from natural language, and one uses vector similarity.
====

[TIP,role=solution]
.Solution
====
**Correct answers:** Cypher Template, Text2Cypher, and Similarity Search.

Aura Agent supports only these three tool types. Cypher Template runs predefined queries with parameters; Text2Cypher generates Cypher from natural language at runtime; Similarity Search finds nodes by vector similarity when embeddings exist. External APIs, stored procedures, and custom functions are not supported.
====
