= Create Your Own Tool
:order: 4
:type: challenge
:disable-cache: true

In this challenge, you will add one Cypher Template tool to the Northwind Customer Service agent you created with AI in Module 1.

== Goal

Create a Cypher Template tool that lets the agent answer a question it cannot handle well with the generated tools alone. Choose a question type, write the Cypher, define the parameter, and test it.

== Before You Start

Open your Northwind Customer Service agent in the Aura Console. Click **Add Tool** → **Cypher Template**.

== Your Task

Add one Cypher Template tool. You decide:

* **What question it answers** (for example: get a customer by company name, list products in a category, or top N customers by order count)
* **The parameter** (name, type, description)
* **The Cypher query** (use `$parameter_name` for the value the LLM will extract)

**Guidelines:**

* The tool description tells the LLM when to use it. Be specific about the question type.
* The parameter description tells the LLM what value to extract. Include an example (for example, "The Northwind customer ID, for example ALFKI or QUICK").
* Test your Cypher in the Query tool first if you are unsure about the schema.

== Test Your Tool

After saving and clicking **Update agent**, ask a question that should trigger your new tool. Expand the **Thought** section to confirm the agent selected it and passed the correct parameter value.
