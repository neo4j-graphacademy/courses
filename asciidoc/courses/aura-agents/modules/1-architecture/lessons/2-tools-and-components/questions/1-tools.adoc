[.question]
= Choosing the Right Tool

For a question like "List products in the Beverages category," which tool should the agent use?

* [ ] Text2Cypher
* [x] Cypher Template
* [ ] Similarity Search
* [ ] All of the above

[TIP,role=hint]
.Hint
====
Cypher Template is best when the query pattern is predictable and you know the parameters, for example category name.
====

[TIP,role=solution]
.Solution
====
**Cypher Template.**

Predictable queries with known parameters, like category name, are best handled by a Cypher Template. The LLM extracts the parameter value and runs your predefined query.
====
