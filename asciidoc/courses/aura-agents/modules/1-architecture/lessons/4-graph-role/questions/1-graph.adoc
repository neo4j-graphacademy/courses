= Graph in Agent Architecture

How does Neo4j serve agents?

* [ ] As a vector store
* [x] As the knowledge base for structured, connected data that tools query
* [ ] Only for logging
* [ ] Only as a cache

[TIP,role=hint]
.Hint
====
The graph stores structured data and relationships that agents query with Cypher or GraphRAG. Aura Agent does not persist conversation history.
====

[TIP,role=solution]
.Solution
====
**As the knowledge base for structured, connected data that tools query.**

Neo4j stores structured, connected data that agents query with Cypher and GraphRAG. Your application can store facts or context in the graph; Aura Agent does not persist conversation history.
====
