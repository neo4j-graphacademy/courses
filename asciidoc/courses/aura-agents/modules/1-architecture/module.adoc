== Module Overview

In this module, you will learn the purpose of agents, the Aura Agent implementation, and the core components.

You will learn:

* The purpose of Aura Agents and when to use them
* The Aura Agent implementation: no/low-code platform, lifecycle, and how to invoke agents
* Core components, primarily the tools available: Cypher Template, Text2Cypher, Similarity Search, plus LLM, knowledge graph, and orchestrator
