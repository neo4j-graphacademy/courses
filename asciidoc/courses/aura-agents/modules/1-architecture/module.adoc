= Architecture and Components of Aura Agents
:order: 1

== Module Overview

In this module, you will learn the architecture and components of Neo4j Aura Agents.

You will learn:

* Agentic concepts and their application in Neo4j
* The agent lifecycle: receive, reason, act, observe, respond
* Core components: LLM, tools, knowledge graph, and orchestrator
* How AuraDB data is used to ground agent responses
