= Introduction to Aura Agents
:order: 1

== Module Overview

In this module, you will learn what Aura Agents are, how they work, and which tools are available. You will then build your first agent.

You will learn:

* The purpose of Aura Agents: no/low-code graph-querying agents built into the Aura Console
* The agent lifecycle: how the orchestrator, LLM, and tools coordinate to answer a question
* What each of the three tools does and when to use it: Cypher Template, Text2Cypher, and Similarity Search

After the two lessons, you will complete a hands-on challenge to create your first agent over the Northwind knowledge graph.
