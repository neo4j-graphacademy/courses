= Purpose of the Create with AI Prompt

What is the purpose of the prompt you provide when you use **Create with AI**?

Select the correct answer.

* [ ] It replaces the agent name and description in the Aura Console
* [x] It describes what the agent should do so the system can build tools that match your requirements
* [ ] It connects the agent to a different AuraDB instance
* [ ] It enables External access and the MCP server for the agent

[TIP,role=hint]
.Hint
====
The lesson defines a prompt as a description of what the agent should do. The agent uses your graph schema to build tools that align with that description.
====

[TIP,role=solution]
.Solution
====
**It describes what the agent should do so the system can build tools that match your requirements.**

The prompt tells the agent its role and scope. The agent uses the schema of your connected knowledge graph to build tools that query the parts of the graph relevant to that prompt. Name, description, instance selection, and access mode are separate configuration steps.
====
