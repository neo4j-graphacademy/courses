[.question]
= What an Aura Agent Does

Which of the following best describes what an Aura Agent does when it handles a user question?

Select the correct answer.

* [ ] It creates the knowledge graph from raw data and keeps the graph up to date
* [x] It interprets the user's input, selects and runs tools against the graph, then turns the results into a natural language answer
* [ ] It lets the user choose which tool runs on each question and returns raw graph records only
* [ ] It writes query results back into the graph to update stored data

[TIP,role=hint]
.Hint
====
The lesson lists three steps: interpret user input, execute tools, generate response. Tools return data. They do not create the graph or write back.
====

[TIP,role=solution]
.Solution
====
**It interprets the user's input, selects and runs tools against the graph, then turns the results into a natural language answer.**

That matches how Aura Agents work: determine what the user needs and which tools to use, run those tools against the graph, then build a natural language answer from the data. The graph already exists in AuraDB. Tools are read-only and do not update the graph.
====
