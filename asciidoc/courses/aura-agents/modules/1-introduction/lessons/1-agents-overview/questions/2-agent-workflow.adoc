[.question]
= Role of Tools

What do Aura Agent tools do, and who turns their results into the answer the user sees?

Select the correct answer.

* [ ] Tools choose which other tools to run and return the final answer to the user
* [ ] The user picks the tools. The agent runs them and returns raw graph data
* [x] Tools return data from the graph. The agent chooses when to call them and turns that data into a natural language answer
* [ ] Tools run automatically and write results back into the graph

[TIP,role=hint]
.Hint
====
Tools are read-only and return data. One component decides which tools to call and then uses that data to build the reply.
====

[TIP,role=solution]
.Solution
====
**Tools return data. The agent chooses when to call them and turns that data into a natural language answer.**

The agent interprets the question, selects one or more tools, runs them against the graph, and gets back data. The agent then turns that data into the answer the user sees. Tools do not decide when to run or what to return. They only retrieve data.
====
