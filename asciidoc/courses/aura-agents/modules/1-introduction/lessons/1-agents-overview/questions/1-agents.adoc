[.question]
= When to Use Agents

Support staff need to ask things like "Which customers ordered product X?" in plain English, without writing Cypher or code. What is a good fit?

Select the correct answer.

* [ ] A batch import job that loads CSV data into the graph
* [x] An Aura Agent connected to the graph so users can ask natural language questions
* [ ] Row-level security so each user sees only their data
* [ ] A parameterized Cypher API that the front end calls with product ID

[TIP,role=hint]
.Hint
====
Agents expose graph data through natural language in the Aura Console. The requirement is "ask in plain English without code," not loading data or restricting rows.
====

[TIP,role=solution]
.Solution
====
**An Aura Agent connected to the graph.**

Aura Agents are built for exposing knowledge graph data using natural language without writing application code. Support asks questions; the agent picks tools, runs queries, and returns an answer. Batch import, row-level security, and a custom Cypher API are different use cases and require code or different tooling.
====
