= When to Use Agents

When are Aura Agents most useful?

* [ ] Only for simple key-value lookups
* [x] When queries require multi-hop reasoning across your graph
* [ ] When you need to run batch imports
* [ ] Only for administrative tasks

[TIP,role=hint]
.Hint
====
Agents reason over connected data and can invoke tools. They excel when the task involves traversal and reasoning.
====

[TIP,role=solution]
.Solution
====
**When queries require multi-hop reasoning across your graph.**

Agents are useful for multi-hop reasoning and natural language questions over complex connected data. Your application can combine the agent's graph-backed answers with other data or APIs; the agent itself only queries your graph.
====
