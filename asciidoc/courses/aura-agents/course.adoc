== Course Description

In this course, you will learn what Aura Agents are, create with AI and from scratch, understand the tools it generates, configure and publish for use in applications like Cursor.

== Duration

{duration}


== What you will learn

* What Aura Agents are and how they work
* How to create an agent with AI and explore what it generates
* What the tools do and how to customise your agent by adding your own tools
* How to publish your agent and connect it through MCP
