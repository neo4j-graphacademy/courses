== Course Description

In this course, you will learn how to build and manage Neo4j Aura Agents: agentic systems that query a knowledge graph stored in AuraDB. Aura Agents use your graph data to answer questions; they do not persist or learn from user interactions across sessions.

You will learn:

* The purpose of Aura Agents, the Aura implementation, and core components, primarily the tools
* How to design and implement agents using Cypher Template and Text2Cypher tools, including Create with AI for rapid setup
* How to publish agents and connect them to hosts such as Claude Desktop and Cursor via MCP or the REST API
