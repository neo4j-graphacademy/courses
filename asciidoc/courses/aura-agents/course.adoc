In this course, you will discover Aura Agent, learn how to design and build no/low-code agents, create the tools that power them, and integrate them into your AI applications.

== What you will learn

* Understand what Aura Agent is, when to use it, and how it works
* Design an agent with clear role, scope, and tool descriptions
* Create an agent with AI and extend it with Cypher Template, Text2Cypher, and Similarity Search tools
* Publish your agent and invoke it using REST APIs and MCP
