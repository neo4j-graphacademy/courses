== Course Description

In this course, you will describe what Aura Agents are, create agents with AI and from scratch, explain the tools they use, and publish and connect them for use in applications like Cursor.

== Duration

{duration}


== What you will learn

* Describe what Aura Agents are, when to use them, and how they work
* Create an agent with AI and explore the tools it generates
* Explain what each tool type does and customise your agent by adding your own tools
* Publish your agent and connect it through MCP
