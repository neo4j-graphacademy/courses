== Course Description

In this course, you will discover Aura Agent, build no/low-code agents, learn how to create the the tools that power them, and integrate them into your AI applications.


== Duration

{duration}


== What you will learn

* Understand what Aura Agent is, when to use it, and how it works
* Create an agent with AI and explore the tools it generates
* Explain what each tool type does and customize your agent by adding your own tools
* Publish your agent and invoke it using REST APIs and MCP

To get the most from this course, have an AuraDB instance with data loaded before you start. We also recommend completing the following courses:
