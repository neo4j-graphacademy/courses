== Course Description

In this course, you will learn how to build and publish Neo4j Aura Agents: no/low-code agentic systems that query a knowledge graph stored in AuraDB. Aura Agents retrieve structured data from your graph to answer questions; they do not persist or learn from user interactions across sessions.

You will learn:

* The purpose of Aura Agents, the Aura implementation, and the available tools: Cypher Template, Text2Cypher, and Similarity Search
* How to design and implement agents using Cypher Template and Text2Cypher tools over the Northwind knowledge graph
* How to publish agents and connect them to Cursor through Model Context Protocol (MCP)
