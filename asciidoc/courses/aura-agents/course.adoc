== Course Description

In this course, you will learn how to build and manage Neo4j Aura Agents, agentic systems backed by a knowledge graph stored in AuraDB.

You will learn:

* The architecture and core components of Aura Agents
* How to design and implement agents using Cypher and tools
* How to leverage GraphRAG for advanced retrieval and reasoning
* How to publish agents for use in hosts such as Claude Desktop and Cursor


== Duration

== What you will learn

* The architecture and components of a Neo4j Aura Agent
* How to design and implement simple Aura agents using Cypher Template, Similarity Search, and Text2Cypher tools
* How to leverage GraphRAG for advanced retrieval and reasoning
* How to publish an agent and use it in a host such as Claude Desktop or Cursor
