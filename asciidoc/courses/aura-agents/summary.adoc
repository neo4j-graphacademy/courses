= Course Summary
:pdf-summary:

In this course, you learned how to build and manage Neo4j Aura Agents.

== Module 1: Introduction to Aura Agents

In this module, you learned:

* The purpose of Aura Agents and when to use them
* The three tool types: where each is used and why
* The agent lifecycle: receive, reason, act, observe, respond
* The core components: LLM, tools, knowledge graph, and orchestrator

== Module 2: Tools and Building Agents

In this module, you learned:

* How to set up Northwind and enable tool authentication
* How to design agents with effective prompts and best practices
* How to implement agents with Cypher Template and Text2Cypher tools
* How to build and test an agent in a hands-on challenge

== Module 3: Build and Publish Agents

In this module, you have:

* Built your own custom agent using Northwind or your AuraDB instance
* Published your agent with External access and MCP server enabled
* Connected to your agent from Claude Desktop or Cursor
* Learned when to use MCP vs REST API
