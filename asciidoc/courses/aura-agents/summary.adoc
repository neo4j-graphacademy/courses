= Course Summary
:pdf-summary:

In this course, you learned how to build and manage Neo4j Aura Agents.

== Module 1: Introduction to Aura Agents

In this module, you learned:

* The purpose of Aura Agents and when to use them
* The three tool types: Cypher Template, Text2Cypher, and Similarity Search—where each is used and why
* How agents work: interpret user input, execute tools, generate response
* How to create an agent with AI and explore the output

== Module 2: Building an Agent

In this module, you learned:

* What Cypher Template and Text2Cypher tools do and when to use each
* How to add and edit Cypher Template tools on your agent
* How to design an agent with role, scope, question types, and tool descriptions
* How to configure tools (edit, add, delete) and build an agent from scratch

== Module 3: Publishing Agents

In this module, you have:

* Compared Internal and External access and chosen the appropriate mode
* Published your agent with External access and MCP server enabled
* Connected to your agent from Cursor using the MCP endpoint
