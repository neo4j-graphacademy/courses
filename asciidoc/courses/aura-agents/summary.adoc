= Course Summary
:pdf-summary:

In this course, you learned how to build and manage Neo4j Aura Agents.

== Module 1: Architecture and Components

In this module, you learned:

* Agentic concepts and their application in the Neo4j ecosystem
* The agent lifecycle: decision-making and action-taking
* The core components of an Aura Agent and their roles
* How Neo4j serves as the knowledge base for agents

== Module 2: Design and Implement

In this module, you learned:

* How to set up Northwind with embeddings and a vector index
* How to design Aura agents with effective prompts and best practices
* How to implement simple agents using Cypher Template, Similarity Search, and Text2Cypher tools
* How to leverage GraphRAG for advanced retrieval and reasoning

== Module 3: Hands-on Challenges

In this module, you have:

* Built a knowledge retrieval agent with custom tools and GraphRAG
* Published an agent and used it in Claude Desktop or Cursor
