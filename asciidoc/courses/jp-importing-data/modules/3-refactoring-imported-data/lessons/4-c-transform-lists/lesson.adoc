[IMPORTANT]
このチャレンジのステップは、以前のチャレンジでデータをインポートし、文字列を日付に変換している場合にのみ実行できます。

このチャレンジは2つのステップで構成されています:

. _Movie_ のプロパティをリストに変換
. グラフに格納されている型を表示
