[.question]
= 2. Which scheme should you use when connecting to a Neo4j Aura Database?

- [ ] `aura://`
- [ ] `bolt://`
- [ ] `neo4j://`
- [*] `neo4j+s://`


[TIP,role=hint]
.Hint
====
A _secure_ connection is required to a Neo4j Aura database.
====

[TIP,role=solution]
.Solution
====
The correct answer is `neo4j+s://`.
====
