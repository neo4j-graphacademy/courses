= Neo4j and LangChain
:order: 1

This module provides an introduction to LangChain and how it integrates with Neo4j to build powerful GenAI applications. You'll set up your development environment and explore a simple agent that demonstrates the core concepts.

In this module, you will learn how to integrate Neo4j with LangChain, a popular framework for building applications with large language models. You will set up your development environment and explore how to use Neo4j as a data source for LangChain agents.

In this module, you will:

* Learn about LangChain and its integration with Neo4j
* Set up your development environment
* Explore a simple LangChain agent that uses Neo4j
