= Getting started
:order: 1
:type: lesson

In this course, you will explore some of the options and approaches for importing data into Neo4j.

You will learn about some tools and techniques for importing data into Neo4j.

You will use the link:https://data-importer.neo4j.io/[Neo4j data importer^] to create a graph database of movies from CSV files.

Finally, you will learn how to prepare data for import and the considerations you need to take when importing data into Neo4j.

You don't need anything other than a browser to access and complete this course.
