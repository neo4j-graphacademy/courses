= Add Movie Nodes
:order: 3
:type: challenge


Read the CSV data and add movie nodes to the database.

Apply acquired learning, import data and create nodes




You should also update some of the default property names to be more concise:

[caption=]
.Update property names
|===
| From | To

| `movie_imdbId` | `imdbId`
| `movie_poster` | `poster`
| `movie_tmdbId` | `tmdbId`
| `movie_url` | `url

|===

You can use the `movieId` as the unique identifier for the `Movie` nodes. 
To set the unique identifier, click the _key_ icon next to the `movieId` property.
