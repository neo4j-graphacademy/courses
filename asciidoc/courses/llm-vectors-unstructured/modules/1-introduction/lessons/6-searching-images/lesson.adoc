= Searching images

Find similar movie posters

Understand that embeddings can represent more than just text.


In previous lessons, you have learned how vectors and embeddings can be used to represent data in different formats. You have seen how embeddings can be used to find similar text.

In this lesson, you will explore a dataset of movie posters and see how vectors and embeddings can be used to find similar images.



MATCH (m:Movie{title: "Get Shorty"})

MATCH (m:Movie{title: "Babe"})


CALL db.index.vector.queryNodes('moviePosters', 6, m.posterEmbedding)
YIELD node, score

RETURN node.title, node.poster, score;
