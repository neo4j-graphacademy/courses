= Getting started with GDS
:order: 1

== Module overview

In this module, you will learn:

* What graph data science is and how it differs from traditional data science
* How graph databases enable analytical approaches impossible with tabular data
* The difference between GDS and Aura Graph Analytics
* How to access and deploy Neo4j GDS and Aura Graph Analytics

This foundational introduction prepares you for learning graph projections, algorithm fundamentals, and eventually applying GDS to real-world problems.
