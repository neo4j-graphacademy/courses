== Steps

1. Update the `params` object to use your name.
This step isn't strictly required, just a bit of fun.


2. The Cypher statement is already written for you.
Call the `session.executeRead()` method, passing a callback function to represent the unit of work.


3. In that function you must call the `run()` method on the first parameter passed to the query, using the `cypher` and `params` variables.

4. `await` the results and use `console.log` to check that the code has executed correctly.

5. To add the new node and relationship to the database, click the Debug icon to the left of the window and run **Create Driver** job, or use the integrated terminal window to run the following command: +
