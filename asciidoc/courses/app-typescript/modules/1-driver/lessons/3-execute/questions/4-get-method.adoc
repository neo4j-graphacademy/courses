[.question]
= Which method would you call to get an individual value from a Record?

* [ ] `record['p'].value`
* [*] `record.get('p')`
* [ ] `record.key('p')`
* [ ] `record.p()`
* [ ] `record.value('p')`


[TIP,role=hint]
.Hint
====
Each record has a method for _getting_ a value.
====

[TIP,role=solution]
.Solution
====
To get the `p` value from a record, call the `record.get('p')` method.
====
