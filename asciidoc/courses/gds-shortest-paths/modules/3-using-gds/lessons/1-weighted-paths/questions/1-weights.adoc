[.question]
= 1. Which of the below options should be used to represent a relationship's weight?

* [ ] `String`
* [ ] `List`
* [x] `Number`
* [ ] `Boolean`

[TIP,role=hint]
.Hint
====
The relationship weight could be used to represent cost, distance or time.
====

[TIP,role=solution]
.Solution
====
The answer is **Number**.
====
