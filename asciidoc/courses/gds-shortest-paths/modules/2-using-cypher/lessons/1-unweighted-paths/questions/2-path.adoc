[.question]
= 2. What type of object does the shortestPath() function return?

* [ ] `Node`
* [ ] `Relationship`
* [ ] `Scalar`
* [x] `Path`


[TIP,role=hint]
.Hint
====
The `shortestPath()` function finds the shortest **path** between two nodes and represents them as a set of relationships.
====

[TIP,role=solution]
.Solution
====
The answer is `Path`.
====
