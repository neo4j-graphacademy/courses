= Getting Started
:order: 1

As you progress through this course, you will create a chatbot.
You can download a link:https://github.com/neo4j-graphacademy/llm-chatbot-python[pre-built repository^] of all the starter code, allowing you to concentrate on creating the chatbot, not the scaffolding.

You can complete the courses using an online IDE or install locally on your machine.

At the end of the course, you will have a template you can use for your future Generative AI Projects.

In this first module, you will get the project up and running.

== What you need

To complete the practical activities in this course, you will need:

* Access to link:https://gitpod.io[Gitpod.io] or link:https://www.python.org/[Python^] installed
* An link:https://platform.openai.com[OpenAI API key^]
* A Neo4j sandbox database
