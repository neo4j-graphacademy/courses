== Verify your results

After running the agent, the following Cypher query should return at least one `ReasoningTrace` node:

[source,cypher]
.Check that reasoning traces were written to Neo4j
----
MATCH (t:ReasoningTrace)
RETURN t.task, t.status, t.created_at
ORDER BY t.created_at DESC
LIMIT 5
----

The challenge verifier checks that your Neo4j instance contains the following node labels:

* At least one `ReasoningTrace` node
* At least one `ReasoningStep` node connected to that trace
* At least one `ToolCall` node connected to a step
