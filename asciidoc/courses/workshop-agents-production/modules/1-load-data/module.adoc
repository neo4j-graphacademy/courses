= Load Northwind with Open Model (with data)
:order: 1

In this module, you load the Northwind graph into your Aura instance using a packaged model file and **Open model (with data)** in the Import tool.

You will learn how to:

* Download the Northwind complete graph package
* Create a graph model in the Import tool and open it with data
* Run the import and confirm results
