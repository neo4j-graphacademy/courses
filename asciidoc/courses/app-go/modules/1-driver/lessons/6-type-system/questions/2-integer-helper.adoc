= 2. Relationship Types

What method would you use to access the type of the relationship `actedIn`?

- [ ] `actedIn["type"]`
- [ ] `actedIn.Labels`
- [*] `actedIn.Type`
- [ ] `neo4j.GetRelationshipType(actedIn)`

====
