[.question]
= 3. Get all Remaining Records

Which of the following examples will allow you to extract an array of any remaining results?

* [ ] `result.All()`
* [*] `result.Collect()`
* [ ] `result.Consume()`
* [ ] `result.Remaining()`


[TIP,role=hint]
.Hint
====
In Neo4j, we call a set of results a _collection_.
====

[TIP,role=solution]
.Solution
====
The answer is `result.Collect()`.
====
