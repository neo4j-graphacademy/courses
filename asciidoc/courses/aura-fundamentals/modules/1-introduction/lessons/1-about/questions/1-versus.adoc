= Where do Aura instances run?

Which of the following describes where Aura instances run?

* [ ] On infrastructure you manage yourself.
* [x] On cloud infrastructure managed by Neo4j.
* [ ] Only offline, as self-managed Neo4j databases.


[TIP,role=hint]
.Hint
====
Aura is a fully managed cloud service where Neo4j handles the infrastructure on your behalf.
====

[TIP,role=solution]
.Solution
====
Aura databases run on cloud infrastructure (AWS, GCP, or Azure) that Neo4j manages on your behalf.
Self-managed databases run on infrastructure you provision and maintain yourself.
====
