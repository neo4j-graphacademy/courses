= Tools
:order: 3

In this module, you will work with the Aura console tools to import, query, explore and build dashboards.

By the end of this module, you will be able to:

* Import data into your Aura instance using the Import tool
* Query data using Cypher in the Query tool
* Explore your graph visually using the Explore tool
* Build a dashboard to visualize your data
* Compare the tools and choose the right one for your task
