Neo4j Aura is a fully managed cloud database service, built on top of the Neo4j database. This course will guide you through the fundamentals of using Neo4j Aura.

== Prerequisites

Before starting this course, you should have a basic understanding of graph databases and some familiarity with Neo4j. If you are new to Neo4j, it is recommended to complete the Neo4j Fundamentals course first, and optionally the Cypher Fundamentals.
