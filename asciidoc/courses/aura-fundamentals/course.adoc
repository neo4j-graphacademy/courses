In this course, you will learn how to use Neo4j AuraDB, a fully managed graph database service in the cloud. Neo4j handles infrastructure—servers, backups, security patches, and scaling—so you can focus on working with your graph data.

== What you will learn

* Signing up for Neo4j Aura and creating an AuraDB instance
* Importing data using the Import tool, mapping CSV data to graph nodes and relationships
* Writing Cypher queries to retrieve and filter data from your graph
* Navigating your graph using the Explore tool to discover relationships and patterns
* Building dashboards to visualize your graph data
* Managing backups, optimize costs, and understand instance lifecycle

== Duration

1 hour
