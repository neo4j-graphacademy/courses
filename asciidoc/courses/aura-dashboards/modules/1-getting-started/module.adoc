= Introduction to Neo4j Dashboards
:order: 1


In this module, you will start to explore Neo4j Dashboards, a tool in Aura that helps you visualize graph data and share insights with others.

You will:

* Learn what Neo4j Dashboards are and when to use them
* Add your GraphAcademy Neo4j instance to the console Aura 
* Explore the example movie recommendations dataset
* Open Dashboards in Aura and connect to your instance
* Create your first dashboard with AI
