= Introduction to Neo4j Dashboards
:order: 1

== Module Overview

This module explains the key concepts of Neo4j Dashboards, including its benefits, use cases, and how it simplifies visualizing graph data.

You will learn how to:

* Import your data into Neo4j
* Explore relationships and patterns in your data
* Generate insights from graph data using the AI assistant

[NOTE]
====
This course uses Neo4j Aura for demonstrations, but all features work identically in on-premises Neo4j Enterprise Edition.
====
