= Understanding the Empty Canvas


Why is the Dashboards canvas page empty when you first access it?

* [ ] Because you need to pay extra.
* [x] Because you need to import data first.
* [ ] Because you need to create another account.
* [ ] Because you need to install additional software.

[TIP,role=hint]
.Hint
====
Dashboards visualize data from your Neo4j instance. Without data loaded into the instance, there is nothing to display on the canvas.
====

[TIP,role=solution]
.Solution
====

**Because you need to import data first.**
To visualize data in Neo4j Dashboards, you first need to import data into your instance. Without any data, the dashboards canvas will be empty as there is nothing to display.

====
