[.question]
= Dashboards requirements


Which of the following requirements should you follow when planning to create a dashboard in Neo4j Dashboards?


* [ ] Use Business Critical tier for exploring and learning about dashboards.
* [ ] Create multiple instances to host different dashboards.
* [x] Import data for exploring and learning about dashboards.
* [ ] Use complex Cypher queries to visualize data relationships.

[TIP,role=hint]
.Hint
====
Before creating dashboards, you must have data in your instance. Import data using the provided Cypher script or CSV files.
====

[TIP,role=solution]
.Solution
====
- Use the **Free tier** for exploring and learning about dashboards.
- Use a single instance to host your dashboards.
- **Import data** for exploring and learning about dashboards.
- Use simple Cypher queries to visualize data relationships.
====
