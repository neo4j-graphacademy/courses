= Requirements for creating dashboards

You want to create a dashboard for your movie recommendation system that visualizes movie ratings and genre popularity. What is required before you can build this dashboard?

* [ ] You must upgrade to Business Critical tier
* [ ] You need to create a separate instance for each dashboard
* [x] You need to load the movie recommendations dataset into your instance
* [ ] You must write complex Cypher queries before creating any visualizations

[TIP,role=hint]
.Hint
====
Dashboards visualize data from your Neo4j instance. Before creating dashboards, you must have data loaded. For the movie recommendation system, you need the movie dataset loaded using the provided Cypher script.
====

[TIP,role=solution]
.Solution
====
**You need to load the movie recommendations dataset into your instance.**

Before creating dashboards, you must have data in your instance. For your movie recommendation dashboard, load the movie dataset using the provided Cypher script. 
====
