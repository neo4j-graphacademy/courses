= Dashboard limits on Free tier

You are building a movies dashboard and want separate dashboard pages for different views, such as content team, marketing, and executives. You are on the AuraDB Free tier. How many dashboards can you create on Free tier?

* [ ] 1 dashboard
* [x] 3 dashboards
* [ ] 10 dashboards
* [ ] Unlimited dashboards

[TIP,role=hint]
.Hint
====
Check the Aura tier limits: Free tier has a fixed maximum number of dashboards you can create.
====

[TIP,role=solution]
.Solution
====
**3 dashboards.**

AuraDB Free tier allows up to 3 dashboards. Professional allows up to 25; Business Critical and Virtual Dedicated Cloud allow unlimited dashboards.
====
