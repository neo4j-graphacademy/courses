[.question]
= Creating Multiple Dashboards

You're building dashboards using the Movies dataset and need to create separate dashboards for different stakeholder groups: one for content teams, one for marketing, and one for executives. You're currently on the AuraDB Free tier.

Can you create all three dashboards on the Free tier?

* [ ] Yes, Free tier allows unlimited dashboards
* [x] Yes, Free tier allows 3 dashboards
* [ ] No, Free tier only allows 1 dashboard
* [ ] No, you need at least 10 dashboards for this use case

[TIP,role=hint]
.Hint
====
The Free tier has a specific limit on the number of dashboards you can create. Consider how many separate dashboards you need for different stakeholder groups.
====

[TIP,role=solution]
.Solution
====
**Yes, Free tier allows 3 dashboards, which is exactly what you need.**

AuraDB Free tier allows you to create up to 3 dashboards. For your use case (content teams, marketing, and executives), the Free tier provides sufficient dashboard capacity. If you need more dashboards in the future, AuraDB Professional allows up to 25 dashboards; AuraDB Business Critical and Virtual Dedicated Cloud allow unlimited dashboards.
====
