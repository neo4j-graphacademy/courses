= Accessing Neo4j Dashboards
:type: lesson
:order: 2


== Lesson Overview
In the previous lesson, you learned about the purpose of Neo4j Dashboards and the benefits of using them for data visualization.

In this lesson, you will learn how to:

* Access Neo4j Dashboards
* Create an account
* Select the appropriate tier
* Navigate the Dashboards UI


== Getting Access to Aura Dashboards

To explore the Dashboards UI, you need an Aura account, and an instance to connect to.

**If you already have an instance**, skip to <<Step 4: Connect to an instance>>.

**If you need to create an account and instance**, follow Steps 1-3 below.

== Step 1: Visit the Neo4j Console

Go to link:https://console.neo4j.io[the Neo4j Console^] in your web browser.

On this page, you will see options to sign up for a new account or log in if you already have one.

Use the **Sign Up** button to create a new account.

image::images/4-sign-up-page.png[Sign up page showing the email address]

For more information, see link:https://neo4j.com/docs/aura/classic/auradb/getting-started/create-database/[Create Database documentation^].


== Step 2: Create Your Account

After clicking the **Sign Up** button, you will be directed to the registration form. Fill in your email address and confirm your identity.

image::images/4-verify-identity.png[Verify identity page, showing the code to enter]


== Step 2b: Create a password
Once your email is verified, create a password:

image::images/4-sign-up-email.png[Sign up page prompting to create a password]



== Step 3: Create an Instance

Once you have an account, create a new instance by selecting a tier based on your requirements.

Choose from:

* **AuraDB Free** - For learning and experimentation
* **AuraDB Professional** - For production applications (no credit card required for trial)
* **AuraDB Business Critical** - For enterprise applications requiring high availability
* **AuraDB Virtual Dedicated Cloud** - For strict compliance and security requirements

image::images/4-free-instance-select.png[Create instance page showing the different tiers available]


== Dashboard limits per tier

The number of dashboards available depends on your tier:

[cols="1,1"]
|===
|Tier |Dashboards

|AuraDB Free
|3

|AuraDB Professional
|25

|AuraDB Business Critical
|25

|AuraDB Virtual Dedicated Cloud
|25
|===

image::images/free-tier-dashboards.png[AuraDB Free dashboards,width=600,align=center]

Upgrade your tier at any time to access more dashboards.

image::images/ee-dashboards.png[Paid tier dashboards,width=600,align=center]


[[step-4]]
== Step 4: Connect to an instance

. Log in to your account and go to the **Dashboards** menu
. Click **Connect to instance** and select your instance from the list
+
image::images/dashboards-ui.png[Dashboards UI,width=600,align=center]

. Enter the connection details for your instance
+
image::images/connect-from-dashboards.png[Connect to instance dialog,width=600,align=center]

Find your connection details by clicking the **...** menu next to your instance name and selecting **Inspect**, or use the credentials file downloaded when you created the instance.

For more information, see link:https://neo4j.com/docs/aura/dashboards/[Neo4j Dashboards documentation^].
