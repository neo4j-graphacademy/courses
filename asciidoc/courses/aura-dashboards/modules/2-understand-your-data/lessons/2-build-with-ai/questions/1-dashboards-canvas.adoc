= Prompting AI for Dashboards


Which of the following is required to visualize data in Aura Dashboards using AI?

* [ ] Writing complex Cypher queries.
* [x] Writing natural language descriptions.
* [ ] Installing additional software.
* [ ] Creating multiple Aura instances.


[TIP,role=hint]
.Hint
====
Think about what is required to visualize data in dashboards using AI.
====

[TIP,role=solution]
.Solution
====
To visualize data in Aura Dashboards using AI, you need to provide natural language descriptions of the insights you want to see. The AI will then generate the necessary visualizations based on your descriptions.
====
