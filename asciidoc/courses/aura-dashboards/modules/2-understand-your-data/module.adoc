= Designing dashboards to understand your data
:order: 2


== Module Overview

This module focuses on understanding your movie recommendation data model and designing dashboards that answer stakeholder questions about movies, actors, ratings, and user preferences.

You will learn how to:

* Describe your movie data model to understand what insights to visualize
* Perform exploratory data analysis to identify key metrics for your recommendation dashboards
* Design dashboards that align with stakeholder goals (content teams, marketing, executives)
* Use GenAI to assist in building dashboards that visualize recommendation insights
