= Building advanced dashboards
:order: 3


== Module Overview

In this module, you will learn how to create and customize complex dashboards in Neo4j. 
You will explore the various tools and features available to build insightful visualizations that help you understand your graph data better.

You will learn how to:

* Create dashboard cards to visualize data using natural language prompts and Cypher queries
* Customize dashboard layouts for better data representation
* Visualize aggregated metrics to gain insights from your data

By the end of this module, you will have a comprehensive movie dashboard that provides stakeholders with valuable insights into movie data.
