= Neo4j Dashboards
:categories: administrator:1, beginners:5, foundation:1, aura:1, coredb
:status: draft
:duration: 2 hours
:caption: Learn about data visualisation in Neo4j.
:key-points: Neo4j Dashboards, exploratory data analysis, data import and visualization, exploring relationships in data.

In this course you will learn about Neo4j Dashboards, a tool for visualising and exploring graph data. This course will prepare you to visualize and understand your Neo4j data.

You will learn how to:

* Access Neo4j Dashboards
* Import and visualize data
* Create interactive dashboards using AI and Cypher queries
* Share dashboards with others


== Duration

{duration}

== Prerequisites

Before starting this course, you should have a basic understanding of graph databases and basic knowledge on Neo4j. If you are new to Neo4j, familiarize yourself with the following courses:

* link:/courses/neo4j-fundamentals/[Neo4j Fundamentals course^] - Learn the basics of graph databases and Neo4j
* link:/courses/aura-fundamentals/[Aura Fundamentals course^] - Learn how to create and manage Aura instances
* link:/courses/cypher-fundamentals/[Cypher Fundamentals course^] - Learn the Cypher query language

== What you will learn

* How to create dashboards in Neo4j
* Understand how to structure Cypher queries to visualize graph data to explore relationships between data points
* How to make dashboards interactive
* How to share dashboards with others
