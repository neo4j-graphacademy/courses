[.question]
= Training data strategy

The training data was created by hand-labeling 150 examples, then using an LLM to annotate ~850 more. Why not hand-label all 1,000?

* [ ] LLM annotations are more accurate than human annotations
* [ ] Hand-labeling 1,000 examples would take months
* [x] 150 hand-labelled examples are enough to show the LLM the task via few-shot prompting, and spaCy tolerates the noise in LLM-generated annotations
* [ ] The LLM was free to use

[TIP,role=hint]
.Hint
====
Think about the cost of annotation time vs the tolerance of the model to noisy labels.
====

[TIP,role=solution]
.Solution
====
Hand-labeling is accurate but slow. 150 gold examples provide enough few-shot context for an LLM to annotate the rest at silver standard. The LLM annotations aren't perfect -- some will have wrong boundaries or labels -- but spaCy's training is tolerant to this noise. The volume of data compensates for individual errors.
====
