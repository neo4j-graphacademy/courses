[.question]
= Undertrained NER

The model trained on 20 examples finds entities in roughly the right places but assigns wrong labels -- sender names tagged as SUBJECT, every email tagged as SENDER_EMAIL. What does this tell you?

* [ ] The model architecture is too small for this task
* [ ] The training data has label errors
* [x] 20 examples is enough to learn where entities are, but not enough to learn which label to assign
* [ ] The model needs pretrained word vectors to distinguish labels

[TIP,role=hint]
.Hint
====
The model finds spans correctly but confuses the labels. Is the problem span detection or classification?
====

[TIP,role=solution]
.Solution
====
With only 20 examples, the model learns that names and email addresses are important spans -- but it hasn't seen enough variety to learn the positional patterns that distinguish senders from recipients (who comes after `From:` vs after `To:`). More training data (1,000+ examples) fixes the label confusion.
====
