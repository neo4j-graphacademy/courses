= Dataset structure

The Enron email PDFs in this course contain government processing stamps (CONFIDENTIAL, Case No, Doc No). Why are these important to understand before parsing?

* [ ] They contain metadata that should be imported into the graph
* [ ] They indicate the classification level of each email
* [x] They are boilerplate that must be stripped before parsing -- otherwise they interfere with header detection
* [ ] They are only present on scanned files

[TIP,role=hint]
.Hint
====
Think about what a parser would do if it encountered `Date: 01/15/2003` from the boilerplate stamp vs `Sent: Wed, 17 Oct 2001` from the actual email header.
====

[TIP,role=solution]
.Solution
====
The boilerplate stamps appear on every document and would confuse a parser that looks for header fields by label. `Date: 01/15/2003` in the stamp is the FOIA production date, not the email's sent date. Stripping boilerplate first ensures the parser only sees the actual email content.
====
