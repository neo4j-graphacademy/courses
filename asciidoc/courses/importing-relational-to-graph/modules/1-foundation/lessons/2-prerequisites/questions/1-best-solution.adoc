[.question]
= Choosing an Import Method

Which of the following is true about choosing how to import data into Neo4j?

Select the correct answer.

* [ ] ETL pipelines are always the best option for any data source
* [ ] LOAD CSV is the only method that works with CSV files
* [x] The best method depends on data source, volume, and whether import is one-time or recurring
* [ ] You must use the Data Importer for AuraDB; no other methods are supported

[TIP,role=hint]
.Hint
====
ETL is one option. Others include Data Importer, LOAD CSV, neo4j-admin, and custom apps. Choose based on data source, volume, and whether import is one-time or recurring.
====

[TIP,role=solution]
.Solution
====
The correct answer is **the best method depends on data source, volume, and whether import is one-time or recurring**.

Choose the import method based on: data source (CSV, relational database, API), data volume, update frequency (one-time vs recurring), and transformation complexity. No single method is always best.
====
