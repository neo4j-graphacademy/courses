[.question]
= Analyzing a Relational Schema

Which SQL query would you use to find all foreign key relationships in a PostgreSQL database?

- [ ] `SELECT * FROM pg_tables`
- [ ] `SHOW FOREIGN KEYS`
- [x] Query the `information_schema.table_constraints` table with `constraint_type = 'FOREIGN KEY'`
- [ ] `DESCRIBE TABLE relationships`


[TIP,role=hint]
.Hint
====
PostgreSQL stores constraint information in the `information_schema` tables.
====

[TIP,role=solution]
.Solution
====
To find foreign key relationships in PostgreSQL, query the `information_schema.table_constraints` table and filter by `constraint_type = 'FOREIGN KEY'`. This returns all foreign key constraints defined in the database.
====
