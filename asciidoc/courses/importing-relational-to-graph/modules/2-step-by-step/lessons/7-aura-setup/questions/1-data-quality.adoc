[.question]
= Data Quality Aspects

Which of the following are aspects of data quality to assess before importing? Select all that apply.

* [x] Accuracy
* [x] Completeness
* [ ] Speed of the import process
* [ ] Number of node labels in the graph

[TIP,role=hint]
.Hint
====
Assess whether the source data is correct, complete, and fit for purpose before you import.
====

[TIP,role=solution]
.Solution
====
**Accuracy** and **Completeness** are aspects of data quality.

* **Accuracy** — Data is correct and error-free.
* **Completeness** — All necessary data is present with no missing elements.

Import speed and number of labels are not data quality dimensions; they are outcomes of the import or model design.
====
