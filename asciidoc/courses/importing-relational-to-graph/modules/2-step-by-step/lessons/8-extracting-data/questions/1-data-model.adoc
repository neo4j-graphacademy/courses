= Data Model

Which of the following is true about the graph data model when importing into Neo4j?

Select the correct answer.

* [ ] You must define the full schema in SQL before importing
* [x] You create the data model as you import; Neo4j is schema-optional
* [ ] The Data Importer requires a pre-built graph model file
* [ ] Node labels must match your relational table names exactly

[TIP,role=hint]
.Hint
====
Neo4j is schema-optional and allows you to create nodes and relationships as you import, without a predefined schema.
====

[TIP,role=solution]
.Solution
====
**You create the data model as you import; Neo4j is schema-optional.**

You do not need a pre-existing schema. The Data Importer (or Cypher) lets you define node labels and relationship types during the import. Table names can map to different label names (e.g. PascalCase).
====
