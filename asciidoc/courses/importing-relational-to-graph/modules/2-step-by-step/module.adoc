= Step by step
:order: 2

This module walks you through importing your relational data into Neo4j. Each lesson focuses on **one topic** and includes **one hands-on challenge** so you can apply the step immediately.

Start with the questions that matter for your use case: *What do I need to query? Which entities and connections are first-class?* Use those answers to drive your schema analysis and model design.

You will:

* Analyze the relational schema and design a graph model (nodes, relationships, properties)
* Set up your environment and extract or prepare source data
* Use the Neo4j Data Importer to create nodes and relationships
* Handle complex data (NULLs, types, large datasets) and run a full import

Work through the lessons in order; by the end you will have imported your data (or the Northwind sample) into Neo4j.
