= Surface Themes - Communities
:order: 3

Nobody at AutoFix maintains a list of "repair themes" - but the themes are in the data, waiting to be found.
In this module, you will run community detection over the document links you built and surface the real clusters: ignition misfires, brake judder, sensor faults.


By the end of this module, you will:

* Explain why community detection over shared-key links produces meaningful themes, not noise
* Project the document graph and run the Leiden algorithm with Graph Data Science
* Tune the resolution parameter to control theme granularity
* Build a themes view an agent can reason over for insights no single document contains
