[.verify.slide]
= Validate the Document Graph

Once you have run all five steps, click the **Check Database** button to verify.

verify::[]

[TIP,role=hint]
.Hint
====
Run the steps in order - later steps `MATCH` nodes that earlier steps create.

**Check:**

* Step 2 created 10 `Document` nodes
* Step 3 created 37 `Section` nodes and the `HAS_SECTION` tree
* Step 5 created `LINKS_TO` relationships
====

[TIP,role=solution]
.Solution
====
Run this query to check your counts:

[source,cypher]
.Check the document graph
----
RETURN COUNT { (:Document) } AS documents,
       COUNT { (:Section) } AS sections,
       COUNT { (:Part) } AS parts,
       COUNT { (:DTC) } AS codes,
       COUNT { ()-[:LINKS_TO]->() } AS links
----

You should see 10 documents, 37 sections, 11 parts, 6 codes, and 42 links.

**If verification fails:**

* Re-run the steps in order from Step 1 - each statement is safe to re-run because it uses `MERGE`
* Check for typos in labels - they are case sensitive
====
