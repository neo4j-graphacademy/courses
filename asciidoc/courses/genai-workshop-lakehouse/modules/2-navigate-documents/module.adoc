= Navigate What's There - Trees
:order: 2

Keyword search over the manual portal returns 40 hits that all look right.
In this module, you will load AutoFix's parsed PDFs into a graph - manuals, bulletins, and recall notices as a navigable tree - and link sections that mention the same part number or trouble code.


By the end of this module, you will:

* Model parsed PDF documents as a Document → Section tree
* Extract part numbers and trouble codes as shared-key nodes
* Derive LINKS_TO relationships between sections in different documents that share a key
* Navigate the document graph with Cypher to answer questions keyword search cannot
