= Course Summary

Congratulations on completing AI on Your Lakehouse: Context Comes in Shapes, Not Queries.

You have learned:

* Why agent context is a problem of shapes, not queries - and where vector search and Text2SQL fall short
* How to load parsed PDF documents into a graph as a navigable tree with shared-key links
* How to surface repair themes nobody named - covering and clustering the *whole* document library with Leiden community detection, instead of sampling it
* How to prove a negative - walk the document structure to find the field codes with *no* documentation, the question similarity search cannot answer
* How to federate the BigQuery warehouse with the graph - crossing the document-to-warehouse boundary, with nothing migrated
* How to write multi-hop Cypher that crosses the document-to-table boundary
* How to drive a coding agent to write Cypher from a spec - using the neo4j-cypher-skill instead of hand-writing queries
* How to use the Neo4j CLI for graph reasoning, and make it available to coding agents
