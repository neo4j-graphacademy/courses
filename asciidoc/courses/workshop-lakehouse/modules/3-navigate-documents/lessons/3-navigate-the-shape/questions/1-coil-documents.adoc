= Documents About the Coil

Run the full-text query for part `IC-2042-A`.
Which folders do the matching documents come from?

* [ ] Only the manuals
* [ ] Manuals and bulletins
* [x] Bulletins, manuals, and recalls
* [ ] Only bulletins

[TIP,role=hint]
.Hint
====
Run the "All documents that mention a part" full-text query with `"IC-2042-A"` and look at the `type` column (`doc.area`).
====

[TIP,role=solution]
.Solution
====
The correct answer is **bulletins, manuals, and recalls**.

The full-text hits for `IC-2042-A` span all three folders: the misfire bulletins `tsb-20-501` and `tsb-22-301`, the ignition service manuals, and the coil recall `rc-2021-01`.
The shared part identifier - which lives in the section text, not in any entity node - connects documents that never cite each other.
====
