[.question]
= Brake Guidance

Adapt the full-text query to part `BP-7720` (the front brake pad kit), scoped to the bulletins folder.
Which of these bulletins references it?

* [x] tsb-21-207
* [ ] tsb-20-501
* [ ] tsb-22-301
* [ ] tsb-30-999

[TIP,role=hint]
.Hint
====
Change the search term to `"BP-7720"` and keep only the hits whose owning document is in `technical-library/bulletins`.
====

[TIP,role=solution]
.Solution
====
The correct answer is **tsb-21-207** - "Marlin EV premature front brake pad and rotor wear".

[source,cypher]
----
CALL db.index.fulltext.queryNodes('content_search', '"BP-7720"')
YIELD node, score
WHERE node:Section
MATCH (doc:Document)-[:HAS*]->(node)
WHERE doc.uri STARTS WITH 'technical-library/bulletins'
RETURN DISTINCT doc.id AS bulletin, doc.title AS title
ORDER BY bulletin
----

`BP-7720` shows up in several brake bulletins (`tsb-20-506`, `tsb-21-207`, `tsb-23-218`, `tsb-24-111`, and more) - the front-pad part is the thread tying them together.
The misfire bulletins `tsb-20-501` and `tsb-22-301` are about ignition coils, so they never appear; `tsb-30-999` does not exist.
====
