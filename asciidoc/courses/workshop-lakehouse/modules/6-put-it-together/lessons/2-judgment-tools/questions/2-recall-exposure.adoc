[.question]
= Open Recall Exposure

Running `recall_exposure.py FAL20T20220002`, which open recall is this vehicle exposed to but never had the remedy applied?

* [ ] RC-2021-01, the ignition coil connector recall
* [x] RC-2021-11, the alternator output failure recall
* [ ] RC-2021-04, the EVAP purge valve recall
* [ ] RC-2022-11, the in-tank fuel pump recall

[TIP,role=hint]
.Hint
====
`recall_exposure.py` ignores the presenting complaint - it returns whatever open recalls are in scope for the model and whose remedy part this vehicle never received.
The misfire is not the recall.
====

[TIP,role=solution]
.Solution
====
The correct answer is **RC-2021-11, "Safety Recall - Alternator Output Failure Stall While Driving"**, whose remedy part is `ALT-8810`.

Note the twist: this has _nothing_ to do with the `P0301` misfire in the bay. The recall check is independent of the presenting complaint - it cross-references the model's open recalls against the warehouse repair history and finds a remedy this vehicle never received.
So the finale repairs the coil _and_ bundles an unrelated open safety recall - a connection that existed in the data all along, but only federating the document graph with the live warehouse could surface.
====
