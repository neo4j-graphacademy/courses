= Understanding Microsoft Agent Framework
:order: 1

In this module, you will learn what Microsoft Agent Framework (MAF) is, why teams use a framework for agents, how MAF structures agents, and where you can continue your learning.

You will:

* Understand Microsoft Agent Framework fundamentals, including what problems it solves, the benefits of framework-based development, and where to find official resources
* Learn the architectural separation between agents and tools, explore the Model Context Protocol (MCP) for GraphRAG systems, and identify key workflow features like streaming and human-in-the-loop patterns
* Connect Neo4j's role as a knowledge layer with MCP integration patterns and identify concrete next steps for continued learning beyond this workshop
