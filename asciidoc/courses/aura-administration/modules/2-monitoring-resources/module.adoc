= Monitoring Resources
:order: 2


== Module Overview

Understanding resource utilization is essential for maintaining optimal database performance.

In this module, you will learn how to navigate the Metrics dashboard and monitor key resource metrics that indicate when your instance needs attention.

You will explore the Resources tab, which provides insights into CPU usage, storage consumption, and query rates.


You will learn how to:

* Navigate the Metrics dashboard

* Monitor CPU usage and identify performance bottlenecks

* Track storage consumption and growth trends

* Analyze query rates to understand workload patterns

* Recognize when to scale your instance
