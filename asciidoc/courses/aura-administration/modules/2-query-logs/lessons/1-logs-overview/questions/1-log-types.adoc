[.question]
= Log Types

Which of the following types of logs are available in Aura?

* [ ] Audit logs
* [ ] Entity logs
* [x] Query logs
* [x] Security logs

[TIP,role=hint]
.Hint
====
Aura provides logs for monitoring query execution and security events.
====

[TIP,role=solution]
.Solution
====
The two types of logs available in Aura are:

**Query logs** - provide detailed visibility into every query executed against your database. +
**Security logs** - track authentication and authorization events.
====
