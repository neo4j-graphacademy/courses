= Addressing the issue

What should you do immediately to address the storage issue?

* [ ] Let it fill up and deal with it Monday morning
* [ ] Run database compaction to free up space
* [x] Scale the instance to get more storage capacity
* [ ] Stop the instance and delete the new data


[TIP,role=hint]
.Hint
====
Storage is at 95% and growing rapidly. What action gives you the capacity you need while you investigate the root cause?
====

[TIP,role=solution]
.Solution
====
The correct answer is to **scale the instance to get more storage capacity**.

When storage is at 95% and growing rapidly, you need additional capacity to ensure the instance has enough space to continue operating.
====
