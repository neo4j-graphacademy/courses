= Keyboard Shortcuts Reference
:description: A comprehensive list of keyboard shortcuts available in the GraphAcademy classroom interface.

== Classroom Interface Shortcuts

=== Slide Navigation

==== Open/Close Slides

* `Cmd + K` (Mac) or `Ctrl + K` (Windows/Linux) - Toggle slides view

* `Cmd + /` (Mac) or `Ctrl + /` (Windows/Linux) - Toggle slides view (alternative)

* `Esc` - Close slides view

==== Navigate Between Slides

* `Arrow Left` - Go to previous slide (only when CMD/Ctrl is not pressed)

* `Arrow Right` - Go to next slide (only when CMD/Ctrl is not pressed)

* `1` through `9` - Jump to specific slide number (e.g., press `3` to go to slide 3)

=== Page Navigation

==== Move Between Lessons

* `Cmd + Arrow Right` (Mac) or `Ctrl + Arrow Right` (Windows/Linux) - Go to next lesson/page

* `Cmd + Arrow Left` (Mac) or `Ctrl + Arrow Left` (Windows/Linux) - Go to previous lesson/page

=== Sandbox Controls

==== Toggle Sandbox

* `Cmd + Q` (Mac) or `Ctrl + Q` (Windows/Linux) - Toggle sandbox maximize/restore (only when sandbox is visible)

=== Navigation Sidebar

==== Close Sidebar

* `Esc` - Close navigation sidebar/TOC overlay (when visible)

=== Chat Interface

==== Message Input

* `Enter` - Send message

* `Shift + Enter` - New line in message (without sending)

* `Arrow Up` - Recall last sent message (when input is empty)

==== Stop Generation

* Click stop button or `Esc` during message generation - Stop AI response generation

=== Lesson Completion

==== Auto-Advance

* `Esc` - Cancel auto-advance countdown when lesson is completed

== Neo4j Browser & Workspace Shortcuts

=== Query Execution

==== Run Queries

* `Cmd + Enter` (Mac) or `Ctrl + Enter` (Windows/Linux) - Execute current query

* `Shift + Enter` - Insert new line in query editor (without executing)

==== Query History

* `Cmd + Up` (Mac) or `Ctrl + Up` (Windows/Linux) - Navigate to previous query in history

* `Cmd + Down` (Mac) or `Ctrl + Down` (Windows/Linux) - Navigate to next query in history

* `Arrow Up` - Previous query in history (when cursor is at start of editor)

* `Arrow Down` - Next query in history (when cursor is at end of editor)

=== Editor Controls

==== Manage Editor

* `Cmd + K` (Mac) or `Ctrl + K` (Windows/Linux) - Clear query editor

* `Cmd + A` (Mac) or `Ctrl + A` (Windows/Linux) - Select all text in editor

* `Cmd + /` (Mac) or `Ctrl + /` (Windows/Linux) - Toggle comment on selected lines

==== Frame Management

* `Esc` - Close current frame/result panel

* `Cmd + Backspace` (Mac) or `Ctrl + Backspace` (Windows/Linux) - Clear all frames

=== Navigation

==== Workspace Navigation

* `Tab` - Move focus to next element

* `Shift + Tab` - Move focus to previous element

* `:help keys` - Display all keyboard shortcuts in Neo4j Browser

== Important Notes

[NOTE]
====

* Keyboard shortcuts are disabled when typing in input fields, text areas, or editable content to prevent conflicts.

* Some shortcuts only work in specific contexts (e.g., sandbox shortcuts only work when the sandbox is visible).

* The slides navigation shortcuts (`Arrow Left`/`Arrow Right`) are disabled when `Cmd`/`Ctrl` is pressed to allow page navigation to take priority.
====
