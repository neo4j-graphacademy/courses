To add the GraphAcademy instance to your Aura console:

. Open the link:https://console.neo4j.com[Aura Console^].

. Goto *Instances*, *Self-managed*, and click *Add deployment*.
+ 
image::{cdn-url}/img/courses/shared/aura-self-managed-annotated.png["Self managed page with the add deployment button highlighted"]

. Select *Monitored*, give the instance a name (for example, `My Sandbox`), and click Connect.
+
image::{cdn-url}/img/courses/shared/aura-add-deployment-annotated.png["Monitored option selected with the instance name My sandbox and connect button highlighted"]

. Enter the following connection details for your sandbox:
+
Connection URL:: [copy]#bolt+s://{instance-host}:{instance-boltPort}#
Username:: [copy]#{instance-username}#
Password:: [copy]#{instance-password}#
Database:: [copy]#{instance-database}#

. Click *Next* to add the instance to your Aura console.
+
image::{cdn-url}/img/courses/shared/aura-my-sandbox.png["Your sandbox instance added as a self-managed instance in the Aura console"]

You can now use your instance in the Aura console to complete the course.

*Make a note of your username, password, and database* as you may need them later.

[IMPORTANT]
.Instance expiration
====
GraphAcadmey instances expire after 3 days.
You can return to this lesson and re-add a new instance if it expires during the course.
====
